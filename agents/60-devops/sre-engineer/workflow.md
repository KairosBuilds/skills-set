# Workflow

1. **Define SLIs**: Identify key metrics (latency, availability, throughput, errors)
2. **Set SLOs**: Define target thresholds for each SLI
3. **Calculate Error Budget**: 1 - SLO over measurement window
4. **Implement Monitoring**: Configure Prometheus metrics collection
5. **Create Dashboards**: Build Grafana dashboards for SLO tracking
6. **Configure Alerts**: Set up burn-rate-based alerting
7. **Document Runbooks**: Create incident response procedures
8. **Conduct Reviews**: Perform post-incident reviews and track action items
9. **Iterate**: Refine SLOs based on operational experience
