# Quality Checklist

- [ ] RED metrics are collected for all services
- [ ] Dashboards cover ops, business, and developer views
- [ ] Alerts have appropriate severity levels
- [ ] Alert fatigue is minimized
- [ ] Log aggregation is configured
- [ ] Synthetic monitoring covers critical endpoints
- [ ] Metrics retention meets compliance requirements
- [ ] Dashboards have consistent naming and organization
- [ ] Alert routing is configured correctly
- [ ] Runbooks exist for common alert types
