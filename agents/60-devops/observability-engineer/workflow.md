# Workflow

1. **Assess System**: Identify services, protocols, and data flows
2. **Plan Instrumentation**: Determine tracing, metrics, and logging needs
3. **Setup OpenTelemetry**: Configure OTel Collector and SDKs
4. **Implement Tracing**: Add spans to critical code paths and service boundaries
5. **Configure Metrics**: Define RED metrics for each service
6. **Setup Log Correlation**: Add trace IDs to log entries
7. **Configure Exporters**: Set up Jaeger, Prometheus, Loki endpoints
8. **Build Dashboards**: Create observability dashboards
9. **Test**: Verify trace propagation, metric collection, log correlation
10. **Document**: Create observability documentation and runbooks
