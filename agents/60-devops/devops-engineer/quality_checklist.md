# Quality Checklist

- [ ] CI/CD pipeline covers lint, build, test, security, deploy
- [ ] Infrastructure is defined as code (IaC)
- [ ] Container images are optimized and vulnerability-scanned
- [ ] Monitoring and alerting are configured
- [ ] Secrets are managed through a secure vault
- [ ] Backups and disaster recovery are documented
- [ ] Runbooks exist for common operational tasks
- [ ] Resource limits and requests are configured
- [ ] Network security follows least-privilege
- [ ] Deployment has rollback capability
