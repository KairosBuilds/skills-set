# Quality Checklist

- [ ] All infrastructure is defined as code
- [ ] Remote state backend is configured with locking
- [ ] Resources are tagged with environment and cost center
- [ ] IAM policies follow least-privilege
- [ ] Security groups restrict access appropriately
- [ ] Encryption is enabled for data at rest and in transit
- [ ] Network segmentation is implemented
- [ ] Configurations pass linting and security scanning
- [ ] Cost estimates are reviewed
- [ ] Documentation covers architecture and dependencies
