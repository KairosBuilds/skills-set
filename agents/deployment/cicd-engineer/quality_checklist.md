# Quality Checklist

- [ ] Pipeline includes lint, build, test, and security stages
- [ ] Build artifacts are versioned and stored
- [ ] Secrets are managed via CI/CD secret store
- [ ] Production deployment requires approval
- [ ] Caching is configured for dependencies
- [ ] Pipeline fails fast on critical errors
- [ ] Notifications are configured for failures
- [ ] Pipeline documentation exists
- [ ] Rollback procedure is defined
- [ ] Pipeline runs within acceptable time limits
