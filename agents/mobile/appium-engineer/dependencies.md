# Dependencies — Appium Engineer

## Core
- Appium >= 2.0
- Node.js >= 18 (for Appium server)
- Python >= 3.10 (test language)
- Java >= 11 (for Android tooling)

## Appium Drivers
- @appium/uiautomator2-driver (Android)
- @appium/xcuitest-driver (iOS)
- appium-flutter-driver (Flutter apps)

## Python Libraries
- Appium-Python-Client >= 3.0
- pytest >= 7.x
- allure-pytest (reporting)
- selenium-wire (network interception)
- Appium-Flutter-Finder (Flutter element finding)

## Java Libraries (Alternative)
- java-client (Appium Java client)
- TestNG / JUnit 5
- Maven / Gradle

## CI/CD
- GitHub Actions / GitLab CI
- Docker (Appium in containers)
- BrowserStack / Saucelabs / LambdaTest SDK

## Tools
- Appium Inspector (element inspection)
- Android SDK Platform Tools (adb)
- Xcode + Command Line Tools (iOS)
- WDA (WebDriverAgent) for iOS
- Allure Framework (reporting)

## Utilities
- opencv-python (visual testing)
- pillow (image processing)
- pytest-xdist (parallel execution)
- pytest-rerunfailures (flaky test retry)
- Python-dotenv (capability configuration)
