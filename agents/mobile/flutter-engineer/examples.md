## Example 1: E-Commerce App
```
Task: Build product listing with search, filtering, and cart management
Output: Multi-screen Flutter app with Riverpod state, Dio networking, and shimmer loading
```

## Example 2: Social Media Feed
```
Task: Implement infinite-scrolling feed with images, likes, and comments
Output: Optimized ListView.builder with cached network images, pull-to-refresh, and animated reactions
```

## Example 3: Real-Time Chat
```
Task: Build real-time messaging with WebSocket support
Output: Chat screen with message bubbles, typing indicators, image attachments, and offline message queue
```

```
Task: Create animated onboarding with auth (email + Google Sign-In)
Output: Smooth PageView onboarding, Riverpod auth state, secure token storage
```

```

```
