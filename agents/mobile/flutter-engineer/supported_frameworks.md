# Supported Frameworks — Flutter Engineer

## Core Frameworks
| Framework | Version | Usage |
|-----------|---------|-------|
| Flutter | >= 3.16 | Cross-platform UI toolkit |
| Dart SDK | >= 3.2 | Programming language and runtime |

## State Management
| Framework | Usage |
|-----------|-------|
| Riverpod 2.x | Compile-safe providers, recommended |
| BLoC / flutter_bloc | Event-driven state management |
| Provider | Legacy state management |
| GetX | Lightweight state + routing |
| MobX | Observable-based state |

## Navigation
| Framework | Usage |
|-----------|-------|
| go_router | Declarative routing with deep linking |
| auto_route | Code-generated type-safe routing |
| Navigator 2.0 | Imperative routing (low-level) |

| Supabase | Postgres backend as a service |
