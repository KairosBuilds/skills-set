## UI Frameworks
| Framework | Usage |
|-----------|-------|
| SwiftUI | Declarative UI for iOS 16+ |
| UIKit | Imperative UI for iOS 15 and below |
| AppKit | macOS application development |
| WatchKit | watchOS interface framework |
| TVMLKit | tvOS application development |
