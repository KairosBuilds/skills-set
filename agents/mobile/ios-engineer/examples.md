# Examples — iOS Engineer

## Example 1: SwiftUI Social Feed
```
Task: Build a Twitter-like feed with compose, like, and share
Output: SwiftUI List with pull-to-refresh, AsyncImage, CoreData caching, and share sheet
```

## Example 2: iCloud Notes App
```
Task: Build a notes app synced across devices via CloudKit
Output: SwiftUI with CloudKit sync, SwiftData storage, markdown rendering, and search
```

## Example 3: Fitness App with HealthKit
```
Task: Display health data, track workouts, show charts
Output: HealthKit integration, Workout route map, activity rings, Widget extension
```

## Example 4: Real-Time Chat
```
Task: Build messaging app with WebSocket and push notifications
Output: URLSession WebSocket, CoreData message cache, APNs, SwiftUI messages list
```

## Example 5: E-Commerce Storefront
```
Task: Build product catalog with search, cart, and checkout
Output: SwiftUI with async image loading, StoreKit in-app purchases, universal links
```

## Example 6: Video Player
```
Task: Implement video player with playlist, AirPlay, and Picture-in-Picture
Output: AVPlayer integration, custom controls, PiP support, now playing info center
```

## Example 7: AR Shopping
```
Task: Place virtual furniture in real space with ARKit
Output: ARSCNView with USDZ model placement, raycasting, AR Quick Look sharing
```

## Example 8: Multiplatform App
```
Task: Build iOS + watchOS + iPadOS app with shared code
Output: Shared SwiftUI views, Watch connectivity, Handoff support, responsive iPad layout
```
