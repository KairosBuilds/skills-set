# Examples — Android Engineer

## Example 1: Modern Note-Taking App
```
Task: Build a full-featured note-taking app with offline support
Output: MVVM architecture, Room database, Hilt DI, Compose UI, search, and cloud sync
```

## Example 2: E-Commerce Checkout
```
Task: Implement checkout flow with cart, address form, and payment integration
Output: Compose LazyColumn cart, form validation, Retrofit payment API, StateFlow state management
```

## Example 3: Social Media Feed
```
Task: Build infinite-scrolling feed with images and interactions
Output: Paging 3 with RemoteMediator, Compose LazyColumn, Coil images, pull-to-refresh
```

## Example 4: Real-Time Messaging
```
Task: Implement chat with WebSocket, messages, and file sharing
Output: OkHttp WebSocket client, Room message persistence, WorkManager for uploads
```

## Example 5: Fitness Tracker
```
Task: Track steps, display charts, and sync with Wear OS
Output: Health Services API, Compose charts, Wear OS tile, Firebase sync
```

## Example 6: Video Streaming App
```
Task: Build video player with ExoPlayer, DRM, and offline downloads
Output: ExoPlayer integration, HLS/DASH support, download manager, casting support
```

## Example 7: AR Shopping Experience
```
Task: Place 3D furniture models in real space using ARCore
Output: SceneView integration, model placement, lighting estimation, snapshot sharing
```

## Example 8: Maps & Location
```
Task: Build location-based service with map, markers, and geofencing
Output: Google Maps Compose, Fused Location Provider, Geofencing API, reverse geocoding
```
