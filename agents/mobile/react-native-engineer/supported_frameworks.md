# Supported Frameworks — React Native Engineer

## Core Frameworks
| Framework | Version | Usage |
|-----------|---------|-------|
| React Native | >= 0.73 | Cross-platform mobile application framework |
| Expo SDK | >= 50 | Managed workflow and dev toolchain |
| Hermes Engine | >= 0.12 | JavaScript engine for production builds |

| Framework/Library | Usage |
