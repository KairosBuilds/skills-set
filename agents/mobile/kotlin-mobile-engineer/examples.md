# Examples — Kotlin Mobile Engineer

## Example 1: KMP Login Module
```
Task: Build a cross-platform login module shared between Android and iOS
Output: KMP shared module with Ktor client, token storage, and platform-specific UI wrappers
```

## Example 2: Shared Networking Layer
```
Task: Create a type-safe networking layer with caching and retry
Output: Ktor client in commonMain with expect/actual for engine, serialization, and SSL
```

## Example 3: Cross-Platform Notes App
```
Task: Build a notes app with offline-first architecture
Output: SQLDelight persistence in shared module, Compose Multiplatform UI, Ktor sync
```

## Example 4: Weather App
```
Task: Display weather data with shared business logic and platform UI
Output: Shared ViewModel + repository, Compose on Android, SwiftUI on iOS
```

## Example 5: Real-Time Chat
```
Task: Implement chat with Ktor WebSocket client in shared module
Output: Shared WebSocket client, Android Compose UI, iOS SwiftUI with ObservableObject
```

## Example 6: E-Commerce Shared Logic
```
Task: Share cart logic, product search, and checkout across platforms
Output: Shared use cases with 95% code sharing, platform-specific UI with native feel
```

## Example 7: Biometric Authentication
```
Task: Implement biometric auth with expect/actual pattern
Output: Common interface with Android BiometricPrompt and iOS LocalAuthentication implementations
```

## Example 8: Analytics SDK
```
Task: Build cross-platform analytics tracking SDK
Output: KMP library with Firebase on Android, Firebase + Analytics on iOS, event queue dispatch
```
