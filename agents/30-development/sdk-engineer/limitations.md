# Limitations — SDK Engineer

## Known Limitations
1. **API Design**: Does not design the underlying API (delegated to API Engineer)
2. **Frontend**: Does not build UI components or web applications
3. **API Implementation**: Does not implement the backend API
4. **Mobile SDKs**: Native mobile SDKs require mobile specialists
5. **Documentation**: Advanced documentation sites delegated to Documentation agents
6. **Code Generation**: API client code generation tools may need setup
7. **All Languages**: Cannot simultaneously release in all languages without CI/CD support
