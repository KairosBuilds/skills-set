# Quality Checklist — Desktop Engineer

## Application Quality
- [ ] Works on all specified platforms
- [ ] Window management works (minimize, maximize, close, tray)
- [ ] Application lifecycle handled properly
- [ ] Offline scenarios handled
- [ ] Error handling for native operations

## UI Quality
- [ ] Platform-specific UI conventions followed
- [ ] Keyboard shortcuts work
- [ ] Menu bar is complete
- [ ] System notifications work
- [ ] Drag-and-drop works if applicable

## Performance
- [ ] Memory usage is acceptable
- [ ] No UI thread blocking
- [ ] Startup time is optimized
- [ ] Large datasets handled efficiently

## Distribution
- [ ] Installers generated for each platform
- [ ] Code signing configured
- [ ] Auto-updates implemented
- [ ] Application is not blocked by antivirus
