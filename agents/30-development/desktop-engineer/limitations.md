# Limitations — Desktop Engineer

## Known Limitations
1. **Mobile Platforms**: Does not build mobile apps (native or React Native)
2. **Web Apps**: Does not build web applications (delegated to Frontend Engineer)
3. **Performance**: Electron apps may have high memory usage
4. **Native Features**: Some platform-specific features may require specialist knowledge
5. **Distribution**: App store distribution requires manual setup
6. **Kernel/Driver**: Does not build kernel modules or device drivers
7. **Embedded Systems**: Does not build embedded or IoT desktop applications
