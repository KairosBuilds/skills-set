# Examples — Framework Engineer

## Example: Web API Framework
**Requirements:** Lightweight, opinionated web API framework

**Design:**
1. Core: DI container, routing, middleware pipeline
2. Features: validation, serialization, auth
3. Plugins: database, caching, logging
4. CLI: project scaffolding, code generation
5. Convention: file-based routing, auto-injection

**Implementation:**
- TypeScript-first with full types
- Plugin system with lifecycle hooks
- Middleware pipeline with before/after hooks
- CLI tool for project creation and code gen

**Output:** Complete web framework with CLI and documentation
