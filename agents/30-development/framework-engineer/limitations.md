# Limitations — Framework Engineer

## Known Limitations
1. **Application Development**: Focuses on frameworks, not applications
2. **Domain Specificity**: General-purpose frameworks may need domain adaptation
3. **Adoption**: Framework adoption and migration require coordination across teams
4. **Performance**: Initial framework versions may need optimization
5. **Testing**: Framework testing requires example applications
6. **Ecosystem**: Building a plugin ecosystem takes time beyond initial development
7. **Learning Curve**: Framework documentation and learning resources need additional investment
