# Limitations — Full Stack Engineer

## Known Limitations
1. **Deep Expertise**: May lack deep framework-specific optimization knowledge
2. **Large Features**: Very large features may need coordination with Senior Software Engineer
3. **Security**: Security review should be delegated to Security agents
4. **DevOps**: Infrastructure and deployment delegated to DevOps Engineer
5. **Testing Depth**: E2E tests are implemented but load testing delegated to Performance agents
6. **Design**: UI/UX design is not created from scratch
7. **Mobile**: Native mobile apps delegated to mobile specialists
