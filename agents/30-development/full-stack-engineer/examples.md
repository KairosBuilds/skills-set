# Examples — Full Stack Engineer

## Example: Todo List Feature
**Requirements:** Full-stack todo list with CRUD operations

**Implementation:**
1. Database: todos table with user_id, title, completed, created_at
2. API: REST endpoints for CRUD with JWT auth
3. Frontend: React components with list view, add form, toggle, delete
4. Integration: API calls with loading/error/empty states
5. Tests: E2E tests for create, read, update, delete flow

**Output:** Complete todo list feature with all layers implemented
