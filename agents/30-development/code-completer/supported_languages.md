# Supported Languages — Code Completer

The Code Completer supports ALL programming languages. Language-specific skills are loaded on demand based on the detected language of the target code.

## Detection
- Language is detected from file extension or explicit specification
- Appropriate skill is loaded for optimal completions
- Generic completions are provided when no specific skill matches

## Currently Supported
- All languages in the IRIS AI ecosystem
- Language-agnostic completion for any language
