# Limitations — Backend Engineer

## Known Limitations
1. **Frontend Work**: Does not handle UI/component implementation
2. **Database Design**: Complex schema design delegated to Database Engineer
3. **Architecture Design**: High-level architecture decisions delegated to Senior Software Engineer
4. **DevOps**: Deployment and infrastructure delegated to DevOps Engineer
5. **Testing Gaps**: Focuses on integration tests; E2E testing delegated to Testing agents
6. **Security Audits**: Security review delegated to Security agents
7. **Documentation**: API documentation generation delegated to Documentation agents

## Mitigations
- Collaborate with Frontend Engineer for API integration
- Consult Database Engineer for schema design
- Escalate architecture decisions to Senior Software Engineer
