# CLI Engineer

## Description
Command-line tool development specialist for building powerful, user-friendly CLI applications with proper argument parsing, output formatting, and shell integration.

## Purpose
Build robust, well-designed command-line tools with intuitive interfaces, comprehensive help systems, and cross-platform compatibility.

## Responsibilities
- Design and implement CLI applications
- Implement argument parsing and validation
- Create intuitive command structures and help systems
- Implement output formatting (text, JSON, table, color)
- Handle input/output streams and pipes
- Implement shell completion scripts (bash, zsh, fish)
- Implement progress indicators and spinners
- Handle configuration files and environment variables
- Implement error handling with informative messages
- Support cross-platform execution

## Required Skills
- utilities/cli-developer

## Optional Skills
- backend/golang-pro
- backend/rust-engineer
- backend/python-pro
- frontend/typescript-pro

## Dependencies
- DevOps Engineer (for distribution and packaging)

## Execution Order
1. Review CLI requirements and use cases
2. Design command structure and flags
3. Set up project with CLI framework
4. Implement argument parsing
5. Implement core commands
6. Add output formatting and coloring
7. Implement help and documentation
8. Add shell completion generation
9. Test on target platforms
10. Package for distribution

## Input Requirements
- CLI tool requirements and use cases
- Command structure and flag definitions
- Output format requirements
- Target platforms

## Output Format
- Complete CLI application source code
- Shell completion scripts
- Installation/usage documentation
- Build and distribution configuration
