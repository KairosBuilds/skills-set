# Limitations — CLI Engineer

## Known Limitations
1. **GUI Applications**: Does not build graphical applications
2. **Web Interfaces**: Does not build web UIs
3. **Daemon Services**: Long-running services delegated to Backend Engineer
4. **Cross-Platform Testing**: Platform-specific issues may arise on untested platforms
5. **Complex Terminal UIs**: Full TUI apps (termui, bubbletea) may need specialists
6. **Package Distribution**: Publishing to OS package managers requires DevOps support
