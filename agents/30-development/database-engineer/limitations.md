# Limitations — Database Engineer

## Known Limitations
1. **Application Logic**: Does not implement business logic
2. **API Design**: Does not design API endpoints
3. **UI/Frontend**: No frontend work
4. **Infrastructure**: Database deployment delegated to DevOps Engineer
5. **High Availability**: Complex HA configurations delegated to Database Architect
6. **Data Migration**: Complex data migrations may require manual review
7. **ORM Optimization**: May need Backend Engineer for advanced ORM integration
8. **Data Warehouse**: OLAP/data warehouse design delegated to Data Engineer
