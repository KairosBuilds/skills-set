# Examples — Frontend Engineer

## Example: User Profile Page
**Specification:** User profile page with edit capability

**Implementation:**
1. ProfileHeader component with avatar and name
2. ProfileForm component with validation
3. API integration for GET/PUT /users/:id
4. Loading skeleton for initial load
5. Toast notifications for save success/error
6. Responsive layout (sidebar on desktop, bottom sheet on mobile)
7. Unit tests for form validation
8. Accessibility: form labels, keyboard navigation, ARIA attributes

**Output:** Complete user profile page with edit functionality
