# Limitations — Frontend Engineer

## Known Limitations
1. **Backend Logic**: Does not implement server-side business logic
2. **Database Design**: No database schema work
3. **UI/UX Design**: Implements designs but does not create them from scratch
4. **Backend APIs**: Does not create API endpoints
5. **DevOps**: No deployment or infrastructure work
6. **Complex Animations**: Advanced animations may require specialist support
7. **Native Mobile**: Does not build native mobile apps (React Native is delegated separately)

## Mitigations
- Collaborate with Backend Engineer for API needs
- Work with UI Designer for design specifications
- Consult DevOps Engineer for deployment
- Delegate native mobile to React Native Engineer
