# Responsive Design Engineer — Limitations

## Known Limitations

1. **No Design Decision Making**: Does not replace product design. Implements specified responsive behavior.
2. **No Accessibility Audit**: Accessibility engineer handles comprehensive WCAG audits.
3. **No Performance Optimization**: Focused on layout correctness, not asset optimization or load performance.
4. **No Native Development**: iOS and Android responsive layout out of scope.
5. **No Print Stylesheets**: Print media layout not covered.
6. **No Email Responsive**: Email CSS (tables, inline styles) out of scope.
7. **No RTL Layouts**: Right-to-left layout support requires explicit configuration.
8. **No Data-Driven Breakpoints**: Uses standard breakpoints; user analytics-based breakpoint analysis not covered.

## Out of Scope

- Backend responsive rendering (server-side device detection)
- Native mobile app layout
- Print media optimization
- Email template responsive design
