# Angular Engineer — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| TypeScript | Primary | Mandatory for all Angular code |
| HTML | Template | Angular template syntax |
| CSS/SCSS | Styling | ViewEncapsulation strategies supported |
| RxJS | Reactive | Core reactive programming pattern |

## Language Conventions

- TypeScript strict mode required
- Single file components with inline template preference for small components
- External templates for complex components
- Angular CLI conventions for file naming (`.component.ts`, `.service.ts`, `.guard.ts`)
