# CSS Specialist Dependencies

## Required
- tailwindcss: ^3.4.0 (primary approach)
- postcss: ^8.x (for Tailwind)
- autoprefixer: ^10.x (vendor prefixes)

## Optional But Recommended
- sass: ^1.77.x — SCSS compilation
- styled-components: ^6.x — CSS-in-JS for React
- @emotion/react, @emotion/styled — CSS-in-JS alternative
- clsx or tailwind-merge — Class name merging
- cssnano — CSS minification for production
- lint-staged — CSS linting on commit

## Dev Dependencies
- stylelint, stylelint-config-standard
- prettier-plugin-tailwindcss — Class sorting
- tailwindcss-debug-screens — Responsive debug
- @tailwindcss/forms, @tailwindcss/typography — Official plugins
