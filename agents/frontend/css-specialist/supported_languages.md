# CSS Specialist — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| CSS | Primary | Modern CSS (Grid, Flexbox, Container Queries, Layers) |
| SCSS/SASS | Primary | Variables, mixins, functions, partials |
| HTML | Template | Semantic markup for styling context |
| JavaScript | Configuration | Tailwind config files |
| CSS-in-JS | Styling | styled-components, Emotion |

## Language Conventions

- CSS: Modern layout with Grid and Flexbox; `:where()`, `:is()`, `has()` selectors
- SCSS: `@use` over `@import`; limited nesting; meaningful partial naming
- Tailwind: Configure in `tailwind.config.js`; use `@apply` sparingly for shared component styles
