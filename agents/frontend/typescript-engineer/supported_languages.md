# TypeScript Engineer — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| TypeScript | Primary | All type-level programming |
| JavaScript | Reference | Understands JS for type generation |
| JSON | Schema | tsconfig, package.json types |

## Language Conventions

- TypeScript 5.5+ syntax supported
- `ts` extension for type files and utilities
- `d.ts` for ambient declarations only
- Barrel exports (`index.ts`) for type modules
- JSDoc for complex generic type documentation
