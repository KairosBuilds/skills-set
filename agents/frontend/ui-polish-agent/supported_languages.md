# UI Polish Agent — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| CSS | Primary | Visual effects, transitions, animations |
| TypeScript | Primary | React/Framer Motion code |
| HTML | Template | Markup for visual context |
| SCSS | Supported | Design token configuration |

## Language Conventions

- Tailwind utility classes for most styling
- CSS custom properties for theme values
- Framer Motion `motion` components for animations
- TypeScript for typed motion component props
