# Next.js Engineer — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| TypeScript | Primary | All Next.js projects use TypeScript |
| JavaScript | Supported | ES2023+, JSX |
| CSS/SCSS | Styling | Tailwind, CSS Modules, global CSS |
| HTML | Template | JSX in React components |
| SQL | Data | For database access in Server Components |

## Language Conventions

- TypeScript required for all production Next.js projects
- `tsx` extension for React components
- `ts` extension for utilities, actions, and configurations
- Path aliases configured in `tsconfig.json` (e.g., `@/` for `app/`)
