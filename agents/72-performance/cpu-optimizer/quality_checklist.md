# Quality Checklist

- [ ] CPU profile collected with sampling profiler
- [ ] Flame graph generated and analyzed
- [ ] Hot paths identified with >5% CPU time
- [ ] Algorithmic complexity reviewed
- [ ] Optimization implemented with evidence
- [ ] Before/after flame graphs compared
- [ ] No correctness regression
- [ ] Benchmark shows measurable improvement
- [ ] Trade-off documented (complexity vs speed)
- [ ] Production-likely workload used for profiling
