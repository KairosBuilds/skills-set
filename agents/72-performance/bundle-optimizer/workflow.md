# Workflow

1. Generate bundle analysis report (Webpack Bundle Analyzer / Vite)
2. Identify large dependencies and duplicate modules
3. Check for unused exports and dead code
4. Implement code splitting (route-level, component-level)
5. Replace heavy libraries with lighter alternatives
6. Optimize asset loading (lazy, responsive, compressed)
7. Verify tree-shaking is working correctly
8. Set bundle size budgets in CI
9. Measure Lighthouse/Web Vitals improvement
10. Document bundle composition and size history
