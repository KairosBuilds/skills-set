# Quality Checklist

- [ ] Bundle analysis report generated
- [ ] Large dependencies identified and evaluated
- [ ] Route-level code splitting implemented
- [ ] Tree-shaking verified for all entry points
- [ ] Duplicate dependencies removed
- [ ] Bundle size budgets set in CI
- [ ] Assets are compressed (images, fonts)
- [ ] Non-critical components are lazy loaded
- [ ] Lighthouse performance score improved
- [ ] Before/after size comparison documented
