# Workflow

1. Profile the system end-to-end (CPU, memory, I/O, network)
2. Identify the primary bottleneck using flame graphs and traces
3. Measure baseline performance metrics
4. Hypothesize optimization strategy
5. Implement optimization
6. Re-profile and compare against baseline
7. Document improvement (before/after)
8. If target not met, identify next bottleneck (repeat)
9. Run functional tests to validate correctness
10. Ship optimization with performance report
