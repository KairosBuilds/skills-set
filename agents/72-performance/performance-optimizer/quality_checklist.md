# Quality Checklist

- [ ] Baseline metrics are established before optimization
- [ ] Bottleneck is identified with profiling evidence
- [ ] One change at a time methodology followed
- [ ] Before/after metrics are documented
- [ ] Functional correctness is validated after optimization
- [ ] Trade-offs are documented (speed vs complexity)
- [ ] Impact on p95 and p99 is measured (not just average)
- [ ] No premature optimization without data
- [ ] Production-like environment used for profiling
- [ ] Optimization report is shared with the team
