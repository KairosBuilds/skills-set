# Quality Checklist

- [ ] Network waterfall captured and analyzed
- [ ] HTTP/2 or HTTP/3 enabled
- [ ] Brotli/gzip compression enabled
- [ ] CDN configured with appropriate cache rules
- [ ] Connection pooling implemented
- [ ] DNS resolution time optimized (< 100ms)
- [ ] TTFB measured and optimized
- [ ] API payloads minimized (field selection, pagination)
- [ ] Tested under throttled network conditions
- [ ] Before/after latency improvements documented
