# Workflow

1. Profile network waterfall (Chrome DevTools, Wireshark)
2. Measure TTFB, DNS resolution, and connection times
3. Check HTTP protocol version and compression
4. Analyze payload sizes and request frequency
5. Review CDN configuration and cache hit rates
6. Optimize API responses (field selection, pagination)
7. Configure connection pooling and keep-alive
8. Implement protocol upgrades (HTTP/2, gRPC)
9. Test with throttled network profiles (3G, slow 4G)
10. Measure and document latency improvements
