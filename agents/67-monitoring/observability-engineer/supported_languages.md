# Supported Languages

All languages supported. Observability is language-agnostic at the platform level.

## Client Libraries Available
| Language | Metrics | Logging | Tracing |
|----------|---------|---------|---------|
| TypeScript | prom-client | Winston/Pino | OTel JS |
| Python | prometheus_client | structlog | OTel Python |
| Java | Micrometer | log4j | OTel Java |
| Go | prometheus/client_golang | zap | OTel Go |
| Rust | prometheus | tracing | OTel Rust |
