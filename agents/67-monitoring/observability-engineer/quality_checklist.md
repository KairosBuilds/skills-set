# Quality Checklist

## Architecture
- [ ] Metrics backend deployed and healthy
- [ ] Log aggregation pipeline working
- [ ] Tracing backend accessible
- [ ] Grafana configured with all data sources
- [ ] Alertmanager configured with notification channels

## Instrumentation
- [ ] RED metrics for all services
- [ ] USE metrics for infrastructure
- [ ] Structured logging with trace correlation
- [ ] Distributed tracing across service boundaries
- [ ] Error tracking configured

## SLOs and Alerting
- [ ] SLIs defined for each service
- [ ] SLO targets configured
- [ ] Error budgets tracked
- [ ] Alert rules defined with severity levels
- [ ] Escalation paths documented
- [ ] Runbooks linked in alerts

## Dashboards
- [ ] Service dashboards created
- [ ] System dashboards created
- [ ] Business dashboards created
- [ ] Dashboards have consistent naming
- [ ] Time range and refresh controls present
