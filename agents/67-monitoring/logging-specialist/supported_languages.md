# Supported Languages

All languages supported.

| Language | Library |
|----------|---------|
| TypeScript/JavaScript | Winston, Pino, Bunyan |
| Python | structlog, logging, loguru |
| Java | log4j, logback, SLF4J |
| Go | zap, zerolog, logrus |
| Rust | log, tracing, slog |
| Ruby | logger, semantic_logger |
