# Cloud Architect Supported Languages

Cloud architecture is language-agnostic at the design level.

| Language | Support Level | Notes |
|----------|--------------|-------|
| TypeScript | Full | CDK, Pulumi IaC |
| JavaScript | Full | CDK, Pulumi IaC |
| Python | Full | CDK, Pulumi, Boto3 IaC |
| Go | Full | Pulumi, Terraform providers |
| Java | Full | Cloud SDK |
| C# | Full | Pulumi, Azure SDK |
| HCL | Full | Terraform configuration |
| YAML/JSON | Full | CloudFormation, ARM, K8s manifests |
| Shell/Bash | Full | Cloud CLI automation |
