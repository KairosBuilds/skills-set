# Microservices Architect Dependencies

## Required Dependencies

| Dependency | Purpose |
|------------|---------|
| Microservices Architect skill (architecture/microservices-architect) | Core design capability |
| DDD Reference Guide | Domain-driven design patterns |

## Upstream Agents

| Agent | When |
|-------|------|
| solution-architect | Provides overall solution context |
| system-architect | Provides system-level constraints |

## Downstream Agents

| Agent | When |
|-------|------|
| api-designer | When detailed API contracts are needed |
| event-driven-architect | When event-driven patterns need deeper design |
| database-architect | When per-service database design is needed |
| cloud-architect | When deployment infrastructure is needed |
| architecture-reviewer | For design validation |

## Runtime Dependencies

- Mermaid diagram rendering
- Service dependency visualization
