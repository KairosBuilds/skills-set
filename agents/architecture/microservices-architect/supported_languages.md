# Microservices Architect Supported Languages

Microservices architecture is language-agnostic. Services can be implemented in any language.

| Language | Support Level | Notes |
|----------|--------------|-------|
| TypeScript | Full | NestJS, Express microservices |
| JavaScript | Full | Express, Fastify microservices |
| Go | Full | Common in microservices ecosystem |
| Java | Full | Spring Boot microservices |
| C# | Full | ASP.NET Core microservices |
| Python | Full | FastAPI, Nameko microservices |
| Rust | Full | Performance-critical services |
| Ruby | Full | Rails API microservices |
| Kotlin | Full | Ktor, Spring Boot microservices |
| PHP | Full | Laravel microservices |
| Swift | Full | Vapor server-side Swift |
| Shell/Bash | Full | Infrastructure automation |
