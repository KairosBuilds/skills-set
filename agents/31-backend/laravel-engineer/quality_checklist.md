# Quality Checklist — Laravel Engineer

- [ ] Migrations created for all schema changes
- [ ] Eloquent relationships defined correctly
- [ ] N+1 queries prevented (eager loading)
- [ ] Form requests for validation
- [ ] API resources for response transformation
- [ ] Pest/PHPUnit tests passing
- [ ] Queue jobs configured (if needed)
- [ ] Authentication implemented (Sanctum/Passport)
- [ ] Authorization policies defined
- [ ] Horizon configured for queue monitoring
