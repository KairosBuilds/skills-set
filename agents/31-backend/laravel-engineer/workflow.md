# Workflow ó Laravel Engineer

## Standard Laravel Development Workflow
1. Review requirements and design
2. Set up Laravel project with Sail
3. Define migrations and Eloquent models
4. Implement relationships and scopes
5. Implement service classes
6. Implement controllers and API resources
7. Implement Livewire components (if needed)
8. Write Pest/PHPUnit tests
9. Configure queue jobs and Horizon
