# NestJS Engineer

## Description
NestJS framework specialist for building enterprise-grade Node.js applications.

## Purpose
Implement production-grade NestJS applications with modular architecture, dependency injection, and enterprise patterns.

## Responsibilities
- Implement NestJS applications with modules
- Implement REST and GraphQL endpoints
- Configure dependency injection
- Implement guards, interceptors, pipes, filters
- Write unit and E2E tests with Jest
- Implement TypeORM/Prisma data access
- Configure OpenAPI documentation
- Implement WebSocket gateways
- Implement microservices with NestJS
