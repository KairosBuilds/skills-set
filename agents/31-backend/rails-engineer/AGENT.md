# Rails Engineer

## Description
Ruby on Rails development specialist for building full-featured web applications.

## Purpose
Implement production-grade Rails applications using Hotwire, Active Record, and Rails conventions.

## Responsibilities
- Implement Rails 7+ applications
- Write Active Record models and migrations
- Implement RESTful controllers
- Build Hotwire (Turbo + Stimulus) interfaces
- Configure Action Cable for WebSockets
- Implement background jobs with Sidekiq
- Write RSpec test suites
- Configure Action Mailer for emails
- Implement authentication with Devise
- Optimize Active Record queries
