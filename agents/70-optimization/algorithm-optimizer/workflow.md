# Workflow

1. Review algorithm implementation and identify hot path
2. Analyze current time and space complexity
3. Identify bottlenecks (nested loops, redundant calculations)
4. Research alternative algorithms or data structures
5. Implement optimized version
6. Benchmark both versions with representative inputs
7. Compare complexity and actual performance
8. Verify correctness matches original
9. Document complexity improvement (before/after)
10. Update API or interface if complexity contract changed
