# Workflow

1. **Define Query**: Specify search criteria (language, topic, stars)
2. **Search**: Query GitHub API for matching repositories
3. **Filter**: Apply quality and relevance filters
4. **Analyze**: Examine code, docs, issues, and activity
5. **Assess Health**: Review commit frequency, issue resolution time
6. **Compare**: Evaluate alternatives side-by-side
7. **Output**: Structured research report with recommendations
