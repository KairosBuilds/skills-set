# Examples — Backend Engineer

## Example: User Management API
**Specification:** REST API for user CRUD operations

**Implementation:**
1. User model with email, name, role fields
2. CreateUserDTO, UpdateUserDTO with validation
3. UserService with create, read, update, delete methods
4. UserController with POST/GET/PUT/DELETE /users endpoints
5. JWT authentication on protected endpoints
6. Role-based authorization for admin operations
7. Integration tests for each endpoint

**Output:** Full user management API with authentication and authorization
