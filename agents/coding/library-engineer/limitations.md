# Limitations — Library Engineer

## Known Limitations
1. **Applications**: Does not build full applications (API, frontend, CLI)
2. **SDK Coordination**: Multi-language SDK coordination delegated to SDK Engineer
3. **Documentation Sites**: Full documentation sites delegated to Documentation agents
4. **Performance Optimization**: Deep optimization delegated to Performance Engineer
5. **Framework Development**: Framework design delegated to Framework Engineer
6. **Security Audit**: Security review delegated to Security agents
7. **Cross-Platform Builds**: Native binary compilation may need DevOps support
