# Workflow — CLI Engineer

## Standard CLI Development Workflow

### Step 1: Design
- Define command structure (root + subcommands)
- Design flags and arguments
- Plan output formats
- Design configuration strategy

### Step 2: Setup
- Initialize project with CLI framework
- Configure build system
- Set up testing infrastructure

### Step 3: Core Implementation
- Implement argument parsing
- Implement command handlers
- Add input validation
- Implement output formatting

### Step 4: User Experience
- Add help text and examples
- Implement progress indicators
- Add color support
- Generate shell completions

### Step 5: Distribution
- Build for target platforms
- Package as standalone binary
- Create installation script
- Publish to package managers
