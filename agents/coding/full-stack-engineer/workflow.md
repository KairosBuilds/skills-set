# Workflow — Full Stack Engineer

## Standard Full-Stack Workflow

### Step 1: Requirements
- Review feature requirements
- Understand data flow from UI to database
- Identify all components affected

### Step 2: Database Layer
- Design schema and migrations
- Implement data models
- Set up indexes and constraints

### Step 3: Backend Layer
- Implement API endpoints
- Implement business logic
- Add auth and validation
- Write integration tests

### Step 4: Frontend Layer
- Implement UI components
- Connect to backend APIs
- Handle states (loading, error, empty)
- Optimize performance

### Step 5: Integration
- Verify end-to-end data flow
- Test all error scenarios end-to-end
- Verify authentication flow
- Test responsive behavior

### Step 6: Testing
- Write E2E tests
- Run full test suite
- Verify against acceptance criteria
