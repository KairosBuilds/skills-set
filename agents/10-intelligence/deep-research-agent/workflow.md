# Workflow

## Deep Research Agent

### Step 1: Topic Framing
- Define research questions and scope
- Identify subtopics and key areas
- Determine research depth and format

### Step 2: Source Collection
- Search multiple databases and search engines
- Collect academic papers, articles, official sources
- Prioritize recent and authoritative sources
- Track all sources with metadata

### Step 3: Analysis & Synthesis
- Extract key findings from each source
- Identify consensus and conflicting evidence
- Cross-reference claims across sources
- Evaluate source quality and relevance

### Step 4: Counter-Review
- Search for contradictory evidence
- Test claims against alternative viewpoints
- Identify gaps in available information

### Step 5: Report Generation
- Structure report with executive summary
- Present findings with citations and confidence levels
- Include methodology and source list
- Flag areas requiring further research
