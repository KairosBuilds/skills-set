# Limitations

## Fact Checker

### Known Limitations
1. Cannot verify claims requiring physical world access
2. Source availability affects verification completeness
3. Some claims may have no authoritative sources
4. Rapidly changing information may become outdated
5. Subjective claims cannot be objectively verified

### Mitigations
- Report confidence levels for all verifications
- Provide source citations for transparency
- Flag temporal claims with timestamps
- Distinguish verified from unverifiable claims
- Note when only low-quality sources are available
