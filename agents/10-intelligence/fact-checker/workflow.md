# Workflow

## Fact Checker

### Step 1: Claim Extraction
- Parse document for factual claims
- Separate verifiable from subjective claims
- Prioritize claims by impact and relevance

### Step 2: Source Identification
- Search for authoritative sources (official docs, peer-reviewed, news)
- Identify primary sources over secondary
- Check source publication dates for freshness

### Step 3: Verification
- Cross-reference claim against multiple sources
- Evaluate source credibility and authority
- Determine confidence level (verified, disputed, unverifiable)

### Step 4: Correction Proposal
- Draft suggested corrections for inaccurate claims
- Provide source citations for each correction
- Flag claims needing context or qualification

### Step 5: Review
- Present findings for user confirmation
- Note any conflicting evidence
- Document verification methodology
