# Workflow

## Agent Builder

### Step 1: Requirements Gathering
- Understand agent purpose and scope
- Define trigger conditions and patterns
- Identify required skills and tools

### Step 2: Agent Structure
- Create directory following agent conventions
- Write AGENT.md with YAML frontmatter
- Create metadata.yaml, config.json
- Write execution_rules.md and workflow.md

### Step 3: Skill Integration
- Identify and reference applicable skills
- Configure tool access and MCP servers
- Define dependency requirements

### Step 4: Documentation
- Write usage examples and quality checklist
- Document limitations and edge cases
- Create supporting documentation files

### Step 5: Validation
- Validate YAML/JSON frontmatter
- Check all required files are present
- Verify trigger patterns are unique
