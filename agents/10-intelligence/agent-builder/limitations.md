# Limitations

## Agent Builder

### Known Limitations
1. Agent quality depends on clear requirements definition
2. Generated agents may need refinement after initial creation
3. Complex agent behavior may require iterative tuning
4. MCP server integration requires external service availability
5. Agent trigger patterns must not conflict with existing agents

### Mitigations
- Iterate on agent design based on testing feedback
- Validate all frontmatter and configuration
- Test agent triggers against real queries
- Review agent guidance for clarity and completeness
