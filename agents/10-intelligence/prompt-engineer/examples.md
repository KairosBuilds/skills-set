# Examples

## Prompt Engineer

### Example 1: System Prompt for Customer Support AI
```markdown
## System Prompt
You are a helpful customer support agent for Acme Corp.
- Be empathetic and professional
- Provide accurate product information
- Escalate complex issues to human agents
- Never make up information about products
```

### Example 2: Structured Output Prompt
```markdown
Extract the following information from the text and return as JSON:
- product_name: string
- price: number
- currency: string
- in_stock: boolean
```

### Example 3: Chain-of-Thought Reasoning Prompt
```markdown
Analyze the user's problem step by step:
1. Identify the issue type
2. Gather relevant information
3. Determine the root cause
4. Propose a solution
5. Verify the solution addresses the issue
```
