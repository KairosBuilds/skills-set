# Dependencies

## Fine Tuning Specialist

### Required
- Python 3.9+
- PyTorch 2.0+
- Hugging Face Transformers
- PEFT library
- Datasets library (Hugging Face)
- GPU with CUDA support (recommended)

### Optional
- QLoRA: bitsandbytes, accelerate
- TRL library (for RLHF)
- Weights & Biases or MLflow (experiment tracking)

### External Services
- Hugging Face Hub (for base models and uploads)
- GPU compute (local, cloud, or hosted)

### Agent Dependencies
- None
