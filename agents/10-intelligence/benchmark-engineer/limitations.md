# Limitations

## Benchmark Engineer

### Known Limitations
1. Benchmarks only measure specific, narrow capabilities
2. Results may not generalize to real-world usage
3. Benchmark contamination can inflate scores
4. Environment differences make cross-study comparison difficult
5. Statistical significance requires sufficient sample sizes

### Mitigations
- Use diverse benchmark suites covering multiple capabilities
- Report methodology and environment details thoroughly
- Use held-out test sets to reduce contamination risk
- Run benchmarks on consistent hardware/software stack
- Apply appropriate statistical tests
