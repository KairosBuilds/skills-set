# Quality Checklist

## Benchmark Engineer

### Design
- [ ] Metrics clearly defined before execution
- [ ] Sample size determined (min 100)
- [ ] Statistical significance threshold set
- [ ] Benchmark methodology documented

### Execution
- [ ] Environment configuration documented
- [ ] Warm-up runs completed and discarded
- [ ] Random seed fixed for reproducibility
- [ ] Contamination check performed on datasets

### Analysis
- [ ] Mean and standard deviation reported
- [ ] Confidence intervals calculated (95%)
- [ ] Effect sizes reported for comparisons
- [ ] Outliers identified and explained
- [ ] Results reproducible from configuration
