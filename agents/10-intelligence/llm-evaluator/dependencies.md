# Dependencies

## LLM Evaluator

### Required
- Promptfoo (for structured evaluation)
- API access to models being evaluated
- Python 3.9+ or Node.js 18+
- tiktoken (for token counting)

### Recommended
- Statistical analysis libraries (scipy, numpy)
- Chart generation (matplotlib, plotly)

### External Services
- Model API endpoints under evaluation

### Agent Dependencies
- May use Prompt Engineer for test prompt design
