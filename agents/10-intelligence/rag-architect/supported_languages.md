# Supported Languages

## RAG Architect

| Language | Support | Notes |
|----------|---------|-------|
| Python | Full | Primary implementation language |
| TypeScript | Partial | Some framework support |
| JavaScript | Partial | Node.js vector store clients |
| SQL | Full | Metadata filtering queries |
| JSON | Full | Document formats |

Note: Python is the primary language for RAG implementation due to framework availability.
