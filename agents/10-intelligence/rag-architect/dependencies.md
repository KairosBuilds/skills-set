# Dependencies

## RAG Architect

### Required
- Python 3.9+
- LangChain or LlamaIndex
- Vector store (Chroma, Pinecone, Weaviate, Qdrant)
- Embedding model API (OpenAI, Hugging Face, or compatible)

### Recommended
- Reranking model (Cohere, cross-encoder)
- BM25 implementation (rank_bm25, Elasticsearch)
- Document parsing libraries (Unstructured, PyMuPDF)

### External Services
- OpenAI API (or alternative embedding provider)
- Vector store service (if not local)

### Agent Dependencies
- None
