# Quality Checklist

## Prompt Optimizer

### Pre-Optimization
- [ ] Original prompt tested and working
- [ ] Optimization goals defined (tokens, accuracy)
- [ ] Token count measured before optimization

### Validation
- [ ] Optimized prompt produces equivalent output
- [ ] Token savings >= 15%
- [ ] All guardrails preserved
- [ ] Semantic meaning unchanged

### Documentation
- [ ] Token savings documented
- [ ] Changes from original documented
- [ ] Known trade-offs noted
- [ ] Rollback available if needed
