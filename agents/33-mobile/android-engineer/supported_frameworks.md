# Supported Frameworks — Android Engineer

## Core Frameworks
| Framework | Usage |
|-----------|-------|
| Android SDK (API 26-34) | Native Android platform APIs |
| Jetpack Compose | Declarative UI framework |
| Material Design 3 | Design system and components |

## Jetpack Libraries
| Library | Usage |
|---------|-------|
| Jetpack Navigation | Screen navigation and deep linking |
| Jetpack ViewModel | UI state management |
| Jetpack Room | SQLite ORM with compile-time verification |
| Jetpack Paging 3 | Paginated data loading |
| Jetpack WorkManager | Deferred and background tasks |
| Jetpack Hilt | Dependency injection |
| Jetpack DataStore | Key-value and proto data storage |
