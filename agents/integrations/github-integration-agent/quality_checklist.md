# Quality Checklist

## PR Operations
- [ ] PR description clear and complete
- [ ] Related issues linked
- [ ] Reviewers assigned
- [ ] All checks passing
- [ ] Merge conflicts resolved
- [ ] Branch cleaned up after merge

## Issue Management
- [ ] Issues properly labeled
- [ ] Milestone assigned
- [ ] Priority set
- [ ] Linked to related items
- [ ] Status kept up-to-date

## Workflow
- [ ] Workflow runs monitored
- [ ] Failures investigated
- [ ] Stale runs cancelled
- [ ] Notifications configured
- [ ] Secrets properly managed
