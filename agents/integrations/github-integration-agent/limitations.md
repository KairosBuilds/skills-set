# Limitations

1. **No GitHub Enterprise Server support** — GitHub.com only (API differences)
2. **Cannot manage team permissions** — individual repo operations only
3. **No multi-repo bulk operations** — single-repository scope
4. **Rate limited** — subject to GitHub API rate limits
5. **Cannot create/delete organizations** — repository-level access only
6. **No webhook configuration** — cannot manage webhooks
7. **Limited to authenticated user scope** — personal token permissions
