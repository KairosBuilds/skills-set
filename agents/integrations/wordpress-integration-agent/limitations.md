# Limitations

1. **No multisite management** — single WordPress installation focus
2. **Cannot manage hosting** — server configuration not included
3. **No plugin marketplace submission** — cannot publish to WordPress.org
4. **Limited to supported WordPress versions** — requires 5.9+
5. **No VIP/Enterprise support** — WordPress.com VIP not covered
6. **Cannot auto-upgrade WordPress core** — manual upgrade process
7. **No security hardening** — recommends but does not enforce security measures
