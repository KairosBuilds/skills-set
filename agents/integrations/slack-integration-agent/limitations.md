# Limitations

1. **No Slack bot interaction** — cannot respond to messages or commands
2. **No file uploads** — text messages only
3. **Cannot read channel history** — send-only access
4. **Rate limited** — subject to Slack rate limits
5. **No user management** — cannot invite or remove users
6. **No channel creation** — must use existing channels
7. **Cannot delete messages** — send-only capability
