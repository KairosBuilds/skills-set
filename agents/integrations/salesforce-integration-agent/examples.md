# Examples

## Example 1: Apex Trigger
```
Input: "Create trigger to update Account rating based on Opportunity"
Actions:
1. Create AccountRatingTrigger on Opportunity
2. After insert, update, undelete
3. Query related Account
4. Calculate rating based on opportunity amount
5. Update Account rating field
6. Write test (90% coverage)
```

## Example 2: LWC Component
```
Input: "Create account search component"
Actions:
1. Create lwc/accountSearch component
2. Add search input with debounce
3. Wire getRecord for account data
4. Display results in datatable
5. Add error handling (toast)
6. Style with SLDS
```

## Example 3: REST API Integration
```
Input: "Integrate external payment gateway"
Actions:
1. Create Apex REST class
2. Implement callout with named credential
3. Map Salesforce objects to JSON payload
4. Handle response and update records
5. Add error handling for timeouts
6. Write integration test with mock
```
