# Limitations

1. **No Jira Data Center support** — Cloud API only
2. **Cannot modify workflows** — workflow state transitions only
3. **No admin operations** — project configuration changes
4. **Confluence page hierarchy limited** — cannot restructure spaces
5. **No attachment handling** — cannot upload/download attachments
6. **JQL limited to available fields** — custom fields need configuration
7. **Sprint management** — cannot delete or merge sprints
