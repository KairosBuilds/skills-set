# Limitations

1. **No interactive messages** — send-only, no button/action support
2. **Cannot read group messages** — bot webhook is outbound-only
3. **No file upload support** — text and markdown only
4. **No user management** — cannot add/remove group members
5. **Markdown size limit** — 4096 bytes maximum
6. **No multi-webhook management** — single webhook at a time
7. **Cannot schedule messages** — instant send only
