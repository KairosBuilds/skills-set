# Dependencies

## Internal Agents
None required.

## External Tools
- **Shopify CLI**: Theme and app development
- **Shopify Admin API**: Store management
- **Shopify Storefront API**: Headless commerce
- **Hydrogen**: React-based storefront framework
- **Oxygen**: Hosting for Hydrogen stores

## Runtime Requirements
- Shopify store access (API key, password)
- Node.js for Hydrogen/app development
- Theme Kit or Shopify CLI for theme sync
