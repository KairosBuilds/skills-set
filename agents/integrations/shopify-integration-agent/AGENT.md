Builds and maintains Shopify stores — developing custom themes, creating apps, integrating APIs, and optimizing commerce experiences.

- Develop Shopify Liquid themes
- Create custom Shopify apps
- Integrate Storefront API
- Configure checkout and payment
- Manage product and collection data
- Optimize store performance
