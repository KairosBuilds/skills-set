# Examples

## Example 1: Custom Section
```
Input: "Add a featured products carousel section"
Actions:
1. Create sections/featured-carousel.liquid
2. Add schema with product picker settings
3. Implement Swiper.js carousel
4. Style with theme CSS variables
5. Add lazy loading for images
6. Test on mobile and desktop
```

## Example 2: Custom App
```
Input: "Create a product review app"
Actions:
1. Scaffold app with Shopify CLI
2. Set up OAuth with read/write scope
3. Create Review model (MongoDB)
4. Implement Admin API GraphQL mutations
5. Create Storefront API extension
6. Set up webhooks for product updates
7. Deploy to cloud (Fly.io)
```

## Example 3: Hydrogen Storefront
```
Input: "Build headless storefront with Hydrogen"
Actions:
1. Create Hydrogen app with Vite
2. Configure Shopify Storefront API
3. Build product listing page
4. Implement cart with useCart hook
5. Add checkout with Shopify Checkout
6. Deploy to Oxygen
```
