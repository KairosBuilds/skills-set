# Execution Rules

## Theme Development
1. Use Liquid templates with proper section/block architecture
2. Follow Shopify theme best practices (performance, accessibility)
3. Implement responsive design (mobile-first)
4. Optimize assets (images, scripts, styles)
5. Test across browsers and devices

## App Development
1. Use Shopify CLI for app scaffolding
2. Implement OAuth flow for API access
3. Use GraphQL for Admin API queries
4. Handle webhook events properly
5. Implement GDPR compliance endpoints

## Storefront API
1. Use Hydrogen for headless commerce
2. Implement cart and checkout flows
3. Handle customer authentication
4. Optimize GraphQL queries for performance
5. Implement caching strategies

## Testing
1. Test in Shopify preview environment
2. Verify across themes and devices
3. Test checkout flows end-to-end
4. Validate webhook handlers
5. Performance test with Lighthouse
