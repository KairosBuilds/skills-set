# Supported Frameworks

All frameworks supported. Migration patterns are framework-agnostic.

| Framework | Support | Notes |
|-----------|---------|-------|
| Flyway | Full | Versioned migrations |
| Liquibase | Full | XML/YAML/JSON changelogs |
| Alembic | Full | Python migrations |
| Prisma Migrate | Full | TypeScript/JS migrations |
| Rails ActiveRecord | Full | Rails migrations |
| Django ORM | Full | Python manage.py migrate |
