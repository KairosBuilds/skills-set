# Supported Frameworks

| Framework | Support Level | Notes |
|-----------|--------------|-------|
| Git | Native | Branching, tagging, hooks |
| CI/CD (GitHub Actions) | Full | Workflow orchestration |
| semantic-release | Full | Automated versioning and publishing |
| Docker | Full | Container building and registry |
| Kubernetes | Full | Deployment manifests |
| Helm | Full | Chart versioning and releases |
