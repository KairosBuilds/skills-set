- [ ] Version consistent across ALL manifests

- [ ] Clean build passes (from scratch)
- [ ] Full test suite passes
- [ ] Security review completed
- [ ] Rollback plan documented and tested
- [ ] Canary/progressive delivery configured for high-risk releases
- [ ] Post-deploy monitoring confirmed
- [ ] Release notes drafted and reviewed
- [ ] No Friday/holiday releases without explicit authorization
