# Supported Languages

All languages supported. Compatibility checking is language-agnostic at the analysis layer.

## Specific Package Managers
| Language | Package Manager | File |
|----------|----------------|------|
| TypeScript/JavaScript | npm/pnpm/yarn | package.json, package-lock.json |
| Python | pip/poetry | requirements.txt, pyproject.toml |
| Rust | cargo | Cargo.toml, Cargo.lock |
| Go | go mod | go.mod, go.sum |
| Java | maven/gradle | pom.xml, build.gradle |
| Ruby | bundler | Gemfile, Gemfile.lock |
