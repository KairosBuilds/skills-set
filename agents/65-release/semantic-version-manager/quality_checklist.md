# Quality Checklist

## Pre-Calculation
- [ ] Git history available from last tag
- [ ] Commit messages follow conventional format
- [ ] Last release tag exists and is correct
- [ ] Version files synchronized

## During Calculation
- [ ] All commits since last tag analyzed
- [ ] Bump type correctly identified
- [ ] Breaking changes flagged
- [ ] Version string format validated

## Post-Calculation
- [ ] Version updated in all project files
- [ ] Annotated tag created
- [ ] Changelog generated and reviewed
- [ ] Tag pushed to remote
- [ ] Release manager notified
- [ ] Version consistency verified across project
