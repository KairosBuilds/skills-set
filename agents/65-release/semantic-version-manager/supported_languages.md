# Supported Languages

All languages supported. Semantic versioning is language-agnostic.

| Language | Version File | Notes |
|----------|-------------|-------|
| TypeScript/JavaScript | package.json | version field |
| Python | pyproject.toml, setup.cfg | version field |
| Rust | Cargo.toml | version field |
| Go | - | git tag based |
| Java | pom.xml, build.gradle | version field |
| .NET | .csproj | VersionPrefix |
