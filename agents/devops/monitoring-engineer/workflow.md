# Workflow

1. **Identify Metrics**: Determine key RED metrics for each service
2. **Configure Collection**: Set up Prometheus exporters or DataDog agents
3. **Build Dashboards**: Create Grafana dashboards for different stakeholders
4. **Define Alerts**: Create alerting rules with severity levels
5. **Setup Log Aggregation**: Configure log shipping and parsing
6. **Configure Synthetic Monitoring**: Set up uptime checks (optional)
7. **Test**: Verify metrics flow, alerts fire correctly
8. **Document**: Create monitoring documentation and runbooks
