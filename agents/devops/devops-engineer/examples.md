# Examples

## Setup Complete DevOps Pipeline
```bash
opencode devops-engineer setup --project ./my-app --ci github-actions --deploy kubernetes
```

## Optimize Infrastructure
```bash
opencode devops-engineer optimize --infra --cost-analysis
```

## Setup Monitoring
```bash
opencode devops-engineer monitoring --stack prometheus-grafana --alerts pagerduty
```

## Output
Configures complete DevOps workflows including CI/CD, infrastructure, containerization, and monitoring.
