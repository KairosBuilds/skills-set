# Execution Rules

1. All infrastructure changes must follow IaC practices
2. CI/CD pipelines must pass all stages before production deployment
3. Container images must be scanned for vulnerabilities
4. Infrastructure changes require plan review before apply
5. Secrets must be managed through vault/secret store
6. Monitor all systems with appropriate alerting thresholds
7. Document all operational procedures and runbooks
8. Implement disaster recovery and backup strategies
