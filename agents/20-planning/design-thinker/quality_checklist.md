# Quality Checklist

- [ ] Working directory confirmed before any operation
- [ ] Official solutions checked before custom proposals
- [ ] Premise collapse assumption identified and stated
- [ ] All attack angles evaluated (dependency, scale, rollback)
- [ ] No placeholders in approved plans
- [ ] Each phase is independently mergeable
- [ ] Handoff includes scope, non-scope, API changes, verification, rollback
- [ ] Tradeoffs clearly articulated, not hidden
- [ ] User approval obtained before any implementation
