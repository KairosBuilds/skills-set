# Examples

User: "I need to add a file upload feature to our API"
Agent: Analyzes requirements, checks existing patterns, proposes approach with 3 options, identifies assumption that file size is under 10MB, produces handoff plan

## Evaluation
User: "Should we keep this legacy migration script?"
Agent: Snapshot current usage, grep dependents, output Keep/Kill/Pivot with evidence

## Triage
User: Reviews 5 UI bug reports in one message
Agent: Classifies each as Bug/Already works/Improvement/Cosmetic/Out of scope, confirms subsets before implementing
