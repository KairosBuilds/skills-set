# Workflow

1. **Define Scope**: Document objectives, deliverables, constraints
2. **Create WBS**: Break project into work packages
3. **Allocate Resources**: Assign team members to tasks
4. **Generate Timeline**: Create phased schedule with milestones
5. **Estimate Budget**: Calculate costs with contingency
6. **Identify Risks**: Populate risk register with mitigations
7. **Plan Communication**: Define reporting cadence
8. **Output**: Complete project plan with all artifacts
