# Workflow

1. **Requirements Workshop**: Gather stakeholder needs
2. **Define Scope**: Feature boundaries and goals
3. **Write User Stories**: As a/I want/So that format
4. **Specify Requirements**: EARS format functional requirements
5. **Define Acceptance Criteria**: Given/When/Then scenarios
6. **Prioritize**: MoSCoW prioritization
7. **Generate Checklist**: Implementation task checklist
8. **Output**: Complete feature specification package
