# Limitations

1. **Token counts are estimates** — model-specific tokenization varies
2. **No real-time cost tracking** — post-hoc analysis
3. **Cannot optimize single large responses** — entire session budget
4. **No model switching** — cannot downgrade model for budget
5. **Budget enforcement is advisory** — cannot hard-stop over-budget tasks
6. **No multi-session budget** — per-session tracking only
7. **Cost estimates based on list prices** — may not reflect actual billing
