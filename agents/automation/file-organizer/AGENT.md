# File Organizer Agent

Analyzes file structures, suggests organizational improvements, detects redundant files, and maintains consistent naming conventions across projects.

## Responsibilities

- Analyze project file structure
- Suggest logical folder organization
- Detect duplicate or unused files
- Enforce naming conventions
- Generate file organization reports
- Clean up temporary files
