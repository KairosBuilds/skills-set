# Limitations

1. **Cannot verify production behavior** — tests only approximate production
2. **No performance regression detection** — load testing not included
3. **Coverage is quantitative, not qualitative** — 100% coverage ≠ bug-free
4. **No security scan verification** — separate process
5. **Flaky tests may cause false failures** — retry logic limited
6. **Empirical evidence is heuristic** — cannot prove absence of bugs
7. **Build verification only** — no deployment verification
