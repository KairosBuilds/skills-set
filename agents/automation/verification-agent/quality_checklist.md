# Quality Checklist

## Pre-Verification
- [ ] All source files saved
- [ ] Dependencies installed
- [ ] Build configuration verified
- [ ] Test configuration verified

## Verification Execution
- [ ] Build completes without errors
- [ ] Lint passes with zero issues
- [ ] Typecheck passes with zero issues
- [ ] All tests pass (unit, integration, e2e)
- [ ] Coverage meets threshold
- [ ] Empirical evidence provided

## Post-Verification
- [ ] Gates summary generated
- [ ] Failures reported with details
- [ ] Blocked items escalated
- [ ] Auto-fix triggered (if applicable)
- [ ] Verification logged for audit
