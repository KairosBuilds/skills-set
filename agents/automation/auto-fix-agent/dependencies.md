# Dependencies

## Internal Agents
- **verification-agent**: Validates fix correctness
- **context-manager**: Manages edit context and history

## External Tools
- **ESLint/Prettier**: Linting and formatting
- **TypeScript compiler**: Type checking
- **Git**: Merge conflict detection
- **Build tools**: Verification

## Runtime Requirements
- Linter/formatter installed
- TypeScript compiler (for TS projects)
- Git CLI with conflict markers support
