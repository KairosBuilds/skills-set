# Bug Finder — Supported Frameworks

All frameworks are supported. This agent operates at the source code level.

## Web Frameworks
- React, Angular, Vue, Svelte, Next.js, Nuxt, Express, Django, Flask, FastAPI, Spring Boot, ASP.NET, Rails, Laravel

## Mobile Frameworks
- React Native, Flutter, SwiftUI, UIKit, Android SDK

## Testing Frameworks
- Jest, Mocha, pytest, JUnit, RSpec, xUnit, NUnit

## Database Frameworks
- Prisma, TypeORM, Sequelize, SQLAlchemy, Hibernate, Entity Framework

## Other
- All widely-used frameworks across all ecosystems
- Framework-specific bug patterns are recognized
