# Bug Finder — Supported Languages

All languages are supported. This agent operates at the source code level using pattern matching and heuristic analysis that works across language boundaries.

## Fully Supported
- JavaScript / TypeScript
- Python
- Java / Kotlin
- C / C++
- C# / .NET
- Go
- Rust
- Ruby
- PHP
- Swift
- Objective-C
- Scala
- Dart
- Shell (Bash / PowerShell)

## Partial Support
- Assembly
- COBOL
- Fortran
- Haskell
- Erlang / Elixir

Reading patterns and general bug detection works on any language. Framework-specific knowledge enhances detection for fully supported languages.
