# Crash Analyzer — Supported Frameworks

All frameworks are supported. Crash analysis works at the runtime level.

## Web Frameworks
- Express, Django, Spring Boot, ASP.NET, Rails, Laravel, Next.js, Nuxt

## Mobile Frameworks
- React Native, Flutter, SwiftUI, UIKit, Android SDK

## Runtime Environments
- Node.js, Deno, Bun
- JVM (OpenJDK, GraalVM)
- .NET Runtime, Mono
- CPython, PyPy
- V8, SpiderMonkey, JavaScriptCore
- Go Runtime
- Rust Runtime
