# Crash Analyzer — Supported Languages

All languages are supported. Crash analysis works at the runtime level, interpreting signals, exceptions, and crash dumps.

## Fully Supported
- JavaScript / TypeScript
- Python
- Java / Kotlin
- C / C++
- C# / .NET
- Go
- Rust
- Ruby
- PHP
- Swift
- Objective-C
- Scala
- Dart

## Crash Dump Support
- Windows MiniDump (.dmp)
- Linux Core Dump
- Java Heap Dump
- Python Traceback
- .NET Exception Stack
- Go Panic Stack
- JavaScript Error Stack
- Rust Panic Messages
