# Runtime Error Specialist — Supported Languages

All languages are supported. Runtime error analysis works at the execution level.

## Fully Supported
- JavaScript / TypeScript
- Python
- Java / Kotlin
- C / C++
- C# / .NET
- Go
- Rust
- Ruby
- PHP
- Swift
- Objective-C
- Scala
- Dart
- Shell (Bash / PowerShell)

## Runtime Error Types Handled
- Null / Undefined references
- Index / Bounds errors
- Type / Cast errors
- Arithmetic errors
- Resource exhaustion
- State / Operation errors
