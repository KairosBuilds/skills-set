# Edge Case Analyzer — Supported Frameworks

All frameworks are supported. Edge case analysis is language-agnostic.

## Web Frameworks
- React, Angular, Vue, Svelte, Next.js, Nuxt, Express, Django, Flask, FastAPI, Spring Boot, ASP.NET, Rails, Laravel

## API Frameworks
- REST APIs (all frameworks)
- GraphQL (Apollo, Relay)
- gRPC
- WebSocket / Socket.IO

## Data Validation
- Zod, Yup, Joi, Valibot
- Pydantic, Marshmallow
- Jakarta Validation, Hibernate Validator
- Data Annotations (.NET)

## Testing
- Jest, Vitest, Mocha, pytest, JUnit, RSpec
- Property-based testing (QuickCheck, Hypothesis, fast-check)
