# Edge Case Analyzer — Supported Languages

All languages are supported. Edge case analysis is language-agnostic and focused on input domains, boundary conditions, and state spaces.

## Fully Supported
- JavaScript / TypeScript
- Python
- Java / Kotlin
- C / C++
- C# / .NET
- Go
- Rust
- Ruby
- PHP
- Swift
- Objective-C
- Scala
- Dart
- Shell (Bash / PowerShell)

## Analysis Types
- Input domain boundaries
- Type system edge cases
- State space exploration
- API contract analysis
- Null/empty/undefined handling
- Numeric boundary conditions
- String and encoding edge cases
