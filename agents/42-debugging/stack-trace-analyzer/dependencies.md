# Stack Trace Analyzer — Dependencies

## Internal Dependencies
- **Crash Analyzer:** Provides stack traces from crashes
- **Runtime Error Specialist:** Provides exception stack traces
- **Bug Finder:** Helps locate bugs from stack trace clues

## Tool Dependencies
- Stack trace parser (multi-format)
- Source map resolver
- Symbol lookup service
- Frame classification engine

## Runtime Dependencies
- Raw stack trace text
- Source maps (for minified JavaScript)
- Debug symbols (optional, for native code)
