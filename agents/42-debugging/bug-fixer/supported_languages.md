# Bug Fixer — Supported Languages

All languages are supported. Fixes are generated at the source code level.

## Fully Supported
- JavaScript / TypeScript
- Python
- Java / Kotlin
- C / C++
- C# / .NET
- Go
- Rust
- Ruby
- PHP
- Swift
- Objective-C
- Scala
- Dart
- Shell (Bash / PowerShell)

## Partial Support
- Assembly
- COBOL
- Fortran
- Haskell
- Erlang / Elixir

Fix quality is highest for fully supported languages with well-established patterns and idioms.
