# Bug Fixer — Dependencies

## Internal Dependencies
- **Bug Finder:** Requires bug reports with precise locations
- **Crash Analyzer:** Crash context for crash-related fixes
- **Runtime Error Specialist:** Exception context for runtime fixes

## Tool Dependencies
- Diff generation utilities
- Test runner (for verification)
- Language-specific compiler/interpreter

## Runtime Dependencies
- Write access to source files
- Build toolchain for verification
- Test framework for validation
