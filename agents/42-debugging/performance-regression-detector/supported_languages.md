# Performance Regression Detector — Supported Languages

All languages are supported. Performance regression analysis works at the metric and benchmark level.

## Fully Supported
- **JavaScript / TypeScript** — Lighthouse, Web Vitals, bundle analysis
- **Python** — Benchmark harnesses, profiling metrics
- **Java / Kotlin** — JMH benchmarks, JFR metrics
- **Go** — Built-in benchmarks, pprof profiles
- **C++** — Google Benchmark, perf metrics
- **Rust** — Criterion benchmarks, perf metrics
- **C# / .NET** — BenchmarkDotNet

## Metric Types Analyzed
- Startup time
- Execution time / latency
- Memory usage
- CPU utilization
- Network request timing
- Bundle size / asset size
- Database query performance
- API response times
