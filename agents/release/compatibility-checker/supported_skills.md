# Supported Skills

All dependency analysis skills are supported.

| Skill Area | Purpose |
|------------|---------|
| Dependency resolution | npm, pip, cargo, maven |
| Semver analysis | Version comparison, range checking |
| Security scanning | CVE detection in dependencies |
| License compliance | Dependency license validation |

## Activate When
- Dependencies are being updated
- New packages are being added
- Release gate requires compatibility check
- Dependency audit is requested
