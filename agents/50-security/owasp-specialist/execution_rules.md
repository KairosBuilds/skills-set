# Execution Rules
1. Map all user input entry points
2. Test for injection in all input parameters
3. Verify authentication mechanisms are complete
4. Check encryption for sensitive data at rest and transit
5. Validate access control on all protected resources
6. Test for XSS in all output contexts
7. Check deserialization routines for safety
8. Verify proper logging of security events
9. Test for SSRF in URL-fetching functionality
