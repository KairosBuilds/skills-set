# Execution Rules
1. Run SAST scan on all source files
2. Scan for hardcoded secrets and credentials
3. Audit dependency vulnerabilities
4. Check for security misconfigurations
5. Verify authentication and authorization implementations
6. Validate input validation and sanitization
7. Check for secure communication (TLS, HSTS)
8. Report findings with CVSS 3.1 scoring
9. Provide remediation guidance for each finding
10. Never disclose actual secrets or credentials in reports
