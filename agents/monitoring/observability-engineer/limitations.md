# Limitations

1. **No APM agent management** — does not deploy/manage APM agents
2. **Single-cluster focus** — multi-cluster observability requires custom setup
3. **No cost optimization** — does not optimize storage/ingestion costs
4. **No multi-cloud support** — single-cloud observability stack
5. **Limited to open-source tools** — Datadog/NewRelic integration via agent only
6. **No synthetic monitoring** — browser/API check not included
7. **Alert fatigue management** — no built-in deduplication tuning
