# Supported Languages

All languages supported via OpenTelemetry SDK.

| Language | OTel SDK |
|----------|----------|
| TypeScript/JavaScript | @opentelemetry/sdk-node |
| Python | opentelemetry-sdk |
| Java | opentelemetry-java |
| Go | opentelemetry-go |
| Rust | opentelemetry-rust |
| .NET | OpenTelemetry .NET |
