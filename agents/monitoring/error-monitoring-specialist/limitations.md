# Limitations

1. **No root cause auto-analysis** — identifies patterns but not root causes
2. **Cannot auto-fix errors** — alerts and tracks only
3. **Limited to tracked platforms** — one error tracking service at a time
4. **No client-side error monitoring** — server-side focus
5. **Sampling affects accuracy** — lower sample rates may miss intermittent errors
6. **No performance error differentiation** — treats all errors equal severity
7. **Source map required for minified code** — without maps, stack traces are unreadable
