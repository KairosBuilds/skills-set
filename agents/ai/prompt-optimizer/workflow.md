# Workflow

## Prompt Optimizer

### Step 1: Input Analysis
- Parse the existing prompt structure
- Identify verbose sections for compression
- Measure current token count

### Step 2: Compression Strategies
- Remove redundant instructions
- Consolidate multiple constraints
- Optimize few-shot example selection
- Apply concise language patterns

### Step 3: Equivalence Validation
- Run original and optimized prompts
- Compare outputs for semantic equivalence
- Measure token savings percentage

### Step 4: Iterative Refinement
- Tighten further if quality threshold is met
- Restore any elements that caused quality loss
- Final validation pass
