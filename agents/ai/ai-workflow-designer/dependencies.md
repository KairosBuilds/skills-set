# Dependencies

## AI Workflow Designer

### Required
- LLM API access
- Workflow orchestration framework (LangChain, CrewAI)
- Python 3.9+ or TypeScript

### Recommended
- Tool integration libraries (requests, BeautifulSoup, Playwright)
- Vector store (for memory/retrieval steps)
- Logging and monitoring stack

### External Services
- Model API endpoints
- Any tools referenced in workflows (search, databases, APIs)

### Agent Dependencies
- May use RAG Architect, Prompt Engineer, Agent Builder
