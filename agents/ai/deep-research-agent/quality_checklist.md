# Quality Checklist

## Deep Research Agent

### Research Design
- [ ] Research questions clearly defined
- [ ] Scope and boundaries set
- [ ] Search strategy documented
- [ ] Source quality criteria established

### Execution
- [ ] Minimum 5 sources per subtopic
- [ ] Sources are diverse (academic, industry, official)
- [ ] Counter-arguments and alternative views included
- [ ] Source freshness verified (publication dates)
- [ ] Contradictory evidence not suppressed

### Reporting
- [ ] Executive summary included
- [ ] Key findings with confidence levels
- [ ] All claims have source citations
- [ ] Methodology section included
- [ ] Areas needing further research identified
- [ ] Report passes counter-review for accuracy
