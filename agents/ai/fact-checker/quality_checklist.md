# Quality Checklist

## Fact Checker

### Verification
- [ ] Claim is factual and verifiable (not subjective)
- [ ] At least 2 independent sources consulted
- [ ] Sources are authoritative and credible
- [ ] Source publication dates checked for freshness
- [ ] Confidence level assigned and justified

### Reporting
- [ ] All sources cited with accessible URLs or references
- [ ] Verified vs disputed vs unverifiable clearly labeled
- [ ] Correction proposals include supporting evidence
- [ ] Temporal context noted for time-sensitive claims
- [ ] Methodology disclosed for verification
