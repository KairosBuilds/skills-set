# Examples

## Document JavaScript with JSDoc
```bash
opencode code-documenter --path ./src --format jsdoc
```

## Document Python with Sphinx
```bash
opencode code-documenter --path ./src --format sphinx
```

## Document C++ with Doxygen
```bash
opencode code-documenter --path ./src --format doxygen
```

## Output
Adds documentation comments to all functions, classes, and methods in the specified source files.
