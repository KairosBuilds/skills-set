# Supported Languages

| Language | Doc Format |
|----------|-----------|
| JavaScript | JSDoc |
| TypeScript | JSDoc / TSDoc |
| Python | Sphinx / reStructuredText / Google Style |
| Java | Javadoc |
| C/C++ | Doxygen |
| C# | XML Documentation Comments |
| Ruby | YARD / RDoc |
| PHP | PHPDoc |
| Rust | Rustdoc |
| Go | Go Comments |
| Kotlin | KDoc |
| Swift | SwiftDoc |
