# Supported Languages

All languages are supported. The agent auto-detects:
- JavaScript/TypeScript (package.json, tsconfig.json)
- Python (pyproject.toml, setup.py, requirements.txt)
- Rust (Cargo.toml)
- Go (go.mod)
- Java (pom.xml, build.gradle)
- Ruby (Gemfile)
- PHP (composer.json)
- C# (.csproj)
- And any language with a standard package manager manifest
