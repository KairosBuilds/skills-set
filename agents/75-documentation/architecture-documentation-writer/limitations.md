# Limitations

1. Diagrams are inferred from code structure, not runtime observation
2. Complex microservice architectures may produce simplified views
3. Cannot auto-detect all inter-service communication patterns
4. External system integration details require manual input
5. Diagram rendering requires external CLI tools (PlantUML/Mermaid)
6. C4 model support is limited to levels 1-3; code-level (level 4) is not auto-generated
