# Workflow

1. **Parse Query**: Identify technology, version, and specific topic
2. **Search Sources**: Query official docs, MDN, readthedocs, etc.
3. **Filter by Version**: Match documentation version to query
4. **Extract Information**: Pull relevant sections and examples
5. **Cross-Reference**: Verify information across sources
6. **Synthesize**: Combine findings into coherent response
7. **Cite**: Document all sources with URLs
8. **Output**: Structured documentation with citations
