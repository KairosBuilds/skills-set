# Workflow

1. **Define Scope**: Specify technology, domain, and practice area
2. **Search Sources**: Query authoritative references and community standards
3. **Identify Consensus**: Find commonly recommended approaches
4. **Validate**: Cross-reference across multiple authoritative sources
5. **Compile**: Organize practices with examples and rationale
6. **Note Conflicts**: Document disagreements or alternatives
7. **Output**: Structured best practices guide with references
