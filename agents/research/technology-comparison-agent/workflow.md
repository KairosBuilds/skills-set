# Workflow

1. **Identify Candidates**: List technologies to compare
2. **Define Dimensions**: Select comparison criteria
3. **Gather Data**: Collect metrics for each dimension
4. **Score**: Rate each technology per dimension
5. **Weight**: Apply importance weights per use case
6. **Analyze**: Identify strengths, weaknesses, trade-offs
7. **Recommend**: Provide data-driven recommendation
8. **Output**: Structured comparison matrix with analysis
