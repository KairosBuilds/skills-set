# CSS Specialist — Limitations

## Known Limitations

1. **No JavaScript Logic**: Does not write business logic. Pure styling focus only.
2. **No Backend Styling**: Server-rendered email CSS (table-based layouts) out of scope.
3. **No Print CSS Specialization**: Basic print styles only; complex print media layouts out of scope.
4. **No Legacy Browser Support**: Focuses on modern browsers (last 2 versions). IE11 not supported.
5. **No CSS Architecture Consulting**: Does not set up BEM, SMACSS, OOCSS from scratch. Works within existing architecture.
6. **No SVG Animation**: Limited SVG manipulation; refer to Animation Engineer for complex SVG animation.
7. **No Figma Plugin Development**: Does not create Figma plugins or design tool extensions.

## Out of Scope

- Business logic and state management
- Database and API integration
- Server configuration and deployment
- Mobile native styling (SwiftUI, Jetpack Compose)
