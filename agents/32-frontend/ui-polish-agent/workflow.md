# UI Polish Agent Workflow

## Phase 1: Audit
1. Review all interactive elements for missing hover/focus/active states
2. Check spacing consistency across the UI
3. Identify components that lack visual refinement
4. Compare light and dark mode visual quality

## Phase 2: Interactions
1. Add hover transitions to cards, buttons, and clickable elements
2. Add micro-interactions (button press, card lift, link underline)
3. Implement loading state animations (skeleton shimmer, spinner styling)
4. Add page transition animations between routes

## Phase 3: Visual Enhancement
1. Apply glassmorphism to overlays, cards, sidebars
2. Add subtle gradients to backgrounds and accents
3. Polish shadows and elevation hierarchy
4. Add glow effects for emphasis elements
5. Refine typography (leading, tracking, balance)

## Phase 4: Consistency
1. Ensure all similar components share the same visual treatment
2. Verify border-radius consistency
3. Check colour palette adherence
4. Review spacing rhythm across pages
5. Final visual QA pass in both light and dark modes
