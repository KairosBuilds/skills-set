# UI Polish Agent Dependencies

## Required
- tailwindcss: ^3.4.0

## Optional But Recommended
- framer-motion: ^11.x — Micro-interactions and animations
- @tailwindcss/forms — Form styling reset
- clsx or tailwind-merge — Class name composition
- @heroicons/react — Consistent icon set

## Dev Dependencies
- prettier-plugin-tailwindcss — Class sorting
- tailwindcss-debug-screens — Viewport debugging
