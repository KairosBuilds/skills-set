# UI Polish Agent — Limitations

## Known Limitations

1. **No Brand Design**: Does not create brand identities, logos, or design systems. Works within existing design language.
2. **No UX Architecture**: Does not restructure information architecture or user flows. Visual polish only.
3. **No Performance Optimization**: Focus on visual quality; may need performance review for heavy visual effects.
4. **No Print Design**: Visual effects for screen only.
5. **No Animation Heavy Lifting**: Complex timeline animations handled by Animation Engineer agent.
6. **No Accessibility Audit**: Accessibility Engineer handles comprehensive a11y. This agent adds polish that should preserve a11y.
7. **No Icon Creation**: Uses existing icon sets. Does not create custom SVG icons.
8. **No Design Token Creation**: Relies on existing design tokens. Does not establish new design systems.

## Out of Scope

- Brand identity and logo design
- UX research and information architecture
- Performance optimization (may need review for heavy effects)
- Print media visual design
