# Next.js Engineer Dependencies

## Required Runtime
- next: ^14.2.0
- react: ^18.3.0
- react-dom: ^18.3.0
- typescript: ^5.4.0

## Optional But Recommended
- zod — Schema validation for Server Actions and forms
- @tanstack/react-query — Client-side mutations and cache
- next-auth or @auth/core — Authentication
- tailwindcss: ^3.x — Styling
- drizzle-orm or prisma — Database access
- next-intl or next-i18next — Internationalization

## Dev Dependencies
- @types/node, @types/react, @types/react-dom
- eslint, eslint-config-next
- prettier
- vitest, @testing-library/react (for client component tests)
- @playwright/test (for E2E)
