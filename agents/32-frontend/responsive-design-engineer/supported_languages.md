# Responsive Design Engineer — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| CSS | Primary | Grid, Flexbox, Container Queries, Media Queries |
| HTML | Primary | Semantic responsive markup |
| SCSS | Supported | Responsive mixins, breakpoint variables |
| JavaScript | Reference | Responsive behavior helpers |

## Language Conventions

- CSS Grid for macro-layouts, Flexbox for micro-layouts
- `clamp()` for fluid values
- CSS custom properties for breakpoint-aware spacing and typography
- Tailwind responsive prefixes in order: base → sm → md → lg → xl → 2xl
