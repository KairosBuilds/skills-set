# Responsive Design Engineer — Supported Frameworks

| Framework | Version | Status | Notes |
|---|---|---|---|
| Tailwind CSS | 3.4+ | Primary | Responsive utility classes |
| Bootstrap | 5.3+ | Supported | Grid system and breakpoints |
| CSS Container Queries | Baseline 2023 | Primary | Component-level responsiveness |

## Framework-Specific Rules

- **Tailwind**: Mobile-first classes (base then sm/md/lg/xl/2xl); container queries with plugin
- **Bootstrap**: 12-column grid; breakpoints at 576/768/992/1200/1400px
- **Container Queries**: Use for reusable components; fallback to media queries for page layout
