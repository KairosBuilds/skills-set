# Responsive Design Engineer Dependencies

## Required
- tailwindcss: ^3.4.0 (for utility-first responsive workflow)
- postcss: ^8.x

## Optional But Recommended
- @tailwindcss/container-queries — Container query plugin
- sass: ^1.77.x (if using SCSS with custom responsive mixins)
- @artsy/fresnel — Responsive components for React/Next.js

## Dev Dependencies
- prettier-plugin-tailwindcss — Sorts responsive classes
- cypress or playwright — Visual regression testing
- storybook — Component-level responsive preview
