# Animation Engineer — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| TypeScript | Primary | Typed animation configurations |
| JavaScript | Secondary | Animation logic |
| CSS | Styling | CSS transitions, keyframes |
| GLSL | Shaders | Custom vertex/fragment shaders for Three.js |

## Language Conventions

- TypeScript for typed variant objects and animation configs
- CSS custom properties for animation timing variables
- JSX for Framer Motion code
- GLSL for R3F shader materials
