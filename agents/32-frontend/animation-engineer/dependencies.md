# Animation Engineer Dependencies

## Required Runtime
- framer-motion: ^11.0.0 (for React animations)
- react: ^18.0.0 (peer dependency)

## Optional But Recommended
- gsap: ^3.12.x — Complex timeline animations
- @react-three/fiber: ^8.x — React renderer for Three.js
- @react-three/drei: ^9.x — R3F utilities
- three: ^0.160.x — 3D rendering engine
- @react-spring/web: ^9.x — Alternative spring animation library
- motion (for non-React web animations)

## Dev Dependencies
- @types/three — TypeScript types for Three.js
- @types/gsap — TypeScript types for GSAP
