# TypeScript Engineer Dependencies

## Required Runtime
- typescript: ^5.5.0

## Optional But Recommended
- zod: ^3.23.x — Runtime validation matching TypeScript types
- type-fest: ^4.x — Additional utility types
- ts-reset — Better array/object type defaults
- ts-pattern: ^5.x — Pattern matching for discriminated unions
- expect-type — Type-level testing

## Dev Dependencies
- @typescript-eslint/eslint-plugin
- @typescript-eslint/parser
- eslint, prettier
- typescript-plugin-css-modules (if using CSS Modules)
- type-coverage — Measure type safety percentage
