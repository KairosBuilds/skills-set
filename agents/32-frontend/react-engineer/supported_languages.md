# React Engineer — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| TypeScript | Primary | All components must be typed |
| JavaScript | Secondary | JSX/ES2023+ syntax supported |
| CSS/SCSS | Styling | Tailwind preferred, CSS Modules for custom |
| HTML | Template | JSX rendering context |
| GraphQL | Query | Apollo Client, Relay |

## Language Conventions

- TypeScript strict mode required
- Functional components only (no class components)
- JSX file extensions: `.tsx` for TypeScript, `.jsx` for JavaScript
- Path aliases configured in `tsconfig.json`
