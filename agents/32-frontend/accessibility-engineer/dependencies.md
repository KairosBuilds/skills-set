# Accessibility Engineer Dependencies

## Required
- axe-core (npm: @axe-core/playwright or @axe-core/cli)

## Optional But Recommended
- @testing-library/jest-dom — Custom matchers for a11y assertions
- @testing-library/user-event — User-centric test events
- eslint-plugin-jsx-a11y — JSX accessibility lint rules
- react-remove-scroll — Body scroll lock for modals (accessible)
- focus-trap-react — Focus trapping utility
- react-aria-components — Accessible React components from Adobe

## Dev Dependencies
- @axe-core/playwright — Automated aXe testing in Playwright
- storybook-addon-a11y — Accessibility panel in Storybook
- lighthouse — Performance and a11y audits
- pa11y-ci — CI accessibility testing
