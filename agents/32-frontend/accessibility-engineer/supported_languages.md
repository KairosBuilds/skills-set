# Accessibility Engineer — Supported Languages

| Language | Proficiency | Notes |
|---|---|---|
| HTML | Primary | Semantic markup, ARIA attributes |
| CSS | Primary | Focus styles, prefers-reduced-motion, contrast |
| TypeScript | Primary | Keyboard handlers, focus management |
| JavaScript | Secondary | A11y runtime logic |
| ARIA | Primary | WAI-ARIA roles, states, properties |

## Language Conventions

- ARIA attributes use kebab-case (`aria-label`, `aria-describedby`)
- WAI-ARIA 1.2 specification compliance
- HTML semantic elements over ARIA whenever possible
- TypeScript for typed focus management utilities
