# Workflow

1. **Input**: Receive high-level objective or user story
2. **Decompose**: Break into atomic tasks using planner skill
3. **Map Dependencies**: Identify prerequisite relationships
4. **Estimate**: Assign effort, complexity, and risk to each task
5. **Sequence**: Order tasks using topological sort
6. **Validate**: Run plan-checker for consistency and completeness
7. **Output**: Produce structured task plan with metadata
