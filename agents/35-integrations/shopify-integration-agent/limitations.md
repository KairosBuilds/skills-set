# Limitations

1. **No Shopify POS development** — online store focus only
2. **Cannot access financial data** — limited to API permissions
3. **No multi-store management** — single store operations
4. **App Store listing** — cannot manage App Store submissions
5. **No fulfillment integration** — not connected to fulfillment services
6. **Theme Kit limitations** — cannot modify certain theme settings
7. **No custom checkout extensions** — limited to Shopify Checkout
