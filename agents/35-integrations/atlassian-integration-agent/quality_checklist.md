# Quality Checklist

## Jira
- [ ] Issues have clear titles and descriptions
- [ ] Correct issue types assigned
- [ ] Priority and labels set
- [ ] Linked to related issues/epics
- [ ] Status transitions accurate
- [ ] Sprint tracking active

## Confluence
- [ ] Pages have consistent structure
- [ ] Content is up-to-date
- [ ] Links to Jira issues active
- [ ] Page hierarchy logical
- [ ] Obsolete content archived

## Cross-Linking
- [ ] Commits reference Jira issues
- [ ] PRs linked to Jira
- [ ] Documentation reflects current state
- [ ] Status synced between systems
