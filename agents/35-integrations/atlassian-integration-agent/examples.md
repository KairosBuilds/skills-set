# Examples

## Example 1: Jira Issue Creation
```
Input: "Create bug ticket for login failure"
Actions:
1. Create issue: "Login fails with valid credentials on mobile"
2. Type: Bug, Priority: High
3. Description: steps to reproduce, expected vs actual
4. Labels: mobile, auth, regression
5. Assign to @developer
6. Link to epic: "Mobile Auth Overhaul"
```

## Example 2: Sprint Planning
```
Input: "Plan next sprint"
Actions:
1. Sort backlog by priority
2. Calculate velocity (last 3 sprints: 25 pts)
3. Select top 25 pts for sprint
4. Create sprint "Sprint 14" (Mon 14 - Fri 25)
5. Move selected issues to sprint
6. Notify team of sprint goals
```

## Example 3: Confluence Docs
```
Input: "Update API documentation"
Actions:
1. Open Confluence space "DEV"
2. Find "API Reference" page
3. Update endpoint descriptions
4. Add new endpoint: POST /api/v2/orders
5. Link to Jira epic "API v2 Release"
6. Review and publish
```
