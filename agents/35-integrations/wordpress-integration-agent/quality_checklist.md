# Quality Checklist

## Theme
- [ ] WordPress coding standards followed
- [ ] Responsive design
- [ ] Proper enqueuing of assets
- [ ] Translation-ready (_e(), __())
- [ ] Accessibility standards met
- [ ] Browser compatibility verified

## Plugin
- [ ] Unique prefix for all names
- [ ] Input sanitization and output escaping
- [ ] Proper hook usage (actions/filters)
- [ ] Uninstall routine implemented
- [ ] Tested with latest WordPress
- [ ] No hardcoded paths or URLs

## WooCommerce
- [ ] Template overrides minimal
- [ ] Payment gateway tested
- [ ] Shipping configuration verified
- [ ] Product import/export working
- [ ] Cart and checkout flows tested
- [ ] Order management functional
