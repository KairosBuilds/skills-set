# Examples

## Example 1: Custom Post Type Plugin
```
Input: "Create plugin for portfolio custom post type"
Actions:
1. Scaffold plugin: portfolio-manager
2. Register custom post type 'portfolio'
3. Add meta boxes (client, date, URL)
4. Register taxonomy 'project-type'
5. Add shortcode [portfolio]
6. Create Gutenberg block for portfolio grid
7. Test with WordPress 6.5
```

## Example 2: WooCommerce Customization
```
Input: "Add custom product field for dimensions"
Actions:
1. Add product data tab via woocommerce_product_data_tabs
2. Save custom field on product save
3. Display field on frontend product page
4. Add field to cart item data
5. Test checkout flow
```

## Example 3: REST API Endpoint
```
Input: "Create custom API endpoint for mobile app"
Actions:
1. Register REST route: /wp/v2/custom-products
2. Implement permission callback
3. Query products with custom meta
4. Format response with proper schema
5. Add pagination support
6. Test with Postman
```
