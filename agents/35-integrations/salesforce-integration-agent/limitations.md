# Limitations

1. **Governor limits apply** — Apex runtime constraints (10s CPU, 100 SOQL queries)
2. **No sandbox management** — cannot create/refresh sandboxes
3. **Limited to single org** — no cross-org operations
4. **No Experience Cloud development** — community sites not covered
5. **Cannot modify standard objects** — only custom fields/objects
6. **No mobile app development** — Salesforce Mobile SDK not included
7. **Deployment requires permissions** — cannot bypass profile/permission checks
