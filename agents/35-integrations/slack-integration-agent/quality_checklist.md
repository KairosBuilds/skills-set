# Quality Checklist

## Message Quality
- [ ] Subject line clear and actionable
- [ ] Content properly formatted
- [ ] Relevant tags included
- [ ] Links functional
- [ ] Within length limits

## Routing
- [ ] Correct channel selected
- [ ] Appropriate notification type
- [ ] Rate limit respected
- [ ] No duplicate messages

## Reliability
- [ ] Message sent successfully
- [ ] Errors logged
- [ ] Retry mechanism active
- [ ] Delivery confirmed where possible
