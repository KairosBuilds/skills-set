# Workflow

1. Review requirements and acceptance criteria
2. Create test strategy document
3. Design test cases covering functional, integration, E2E, and non-functional
4. Prioritize test execution by risk and impact
5. Execute test suites across all layers
6. Log and track defects with severity classification
7. Verify fixes with regression testing
8. Generate quality metrics reports (pass rate, coverage, defect density)
9. Conduct release readiness assessment
10. Archive test artifacts for compliance
