# Execution Rules

1. Define test strategy before test execution begins
2. Escalate critical defects immediately with full reproduction steps
3. Track all testing activities with traceability to requirements
4. Enforce quality gates at every stage (dev, staging, production)
5. Measure and report quality metrics weekly
6. Balance automation coverage with exploratory testing
7. Maintain a living test plan that evolves with the product
8. Validate acceptance criteria before marking stories complete
9. Conduct root cause analysis on production defects
10. Never compromise quality for schedule
