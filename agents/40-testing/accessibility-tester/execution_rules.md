# Execution Rules

1. Audit against WCAG 2.1 AA minimum (AAA where specified)
2. Run automated scans on every page template
3. Supplement automated scans with manual keyboard testing
4. Test with screen readers (NVDA, VoiceOver, JAWS)
5. Verify color contrast in both light and dark modes
6. Check focus indicators and tab order
7. Validate ARIA usage matches actual semantics
8. Test zoom up to 400% for layout breakage
9. Report violations with element selectors and fix guidance
10. Track accessibility debt across releases
