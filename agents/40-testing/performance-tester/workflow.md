# Workflow

1. Define performance test scenarios and acceptance criteria
2. Set up test environment with monitoring
3. Write test scripts (k6, Artillery, JMeter)
4. Execute baseline test to establish reference
5. Run load test at target throughput
6. Run stress test to find breaking point
7. Run soak test over extended duration
8. Analyze results and identify bottlenecks
9. Generate performance report with recommendations
10. Re-test after optimizations are applied
