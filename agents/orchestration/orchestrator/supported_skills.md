# Orchestrator Supported Skills

The Orchestrator is meta-capable — it does not directly perform domain skills but routes to agents that do.

## Direct Capabilities

| Skill | Description |
|-------|-------------|
| automation/delegation | Task routing and agent delegation |
| automation/planner | Task decomposition and workflow planning |
| automation/verifier | Output validation against acceptance criteria |
| code-quality/code-reviewer | Quality assurance on merged outputs |
| automation/context-compressor | Context management and compression |

## Routing Access (All Agent Skills)

The orchestrator can route to any agent that supports these skill categories:

- All `architecture/*` skills
- All `backend/*` skills
- All `frontend/*` skills
- All `database/*` skills
- All `devops/*` skills
- All `security/*` skills

- All `code-quality/*` skills
- All `productivity/*` skills
- All `automation/*` skills
- All `opencode/*` skills
