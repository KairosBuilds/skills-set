# Quality Checklist

- [ ] System architecture context understood
- [ ] Design decisions evaluated against alternatives
- [ ] Trade-offs documented for each major recommendation
- [ ] Long-term maintenance implications assessed
- [ ] Scalability and performance considerations addressed
- [ ] Test strategy evaluated for adequacy
- [ ] Operational concerns (deployment, monitoring) reviewed
- [ ] Backward compatibility and migration paths considered
- [ ] Recommendations prioritized and actionable
- [ ] Report structured with clear severity classification
