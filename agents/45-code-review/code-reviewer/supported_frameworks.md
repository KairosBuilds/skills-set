- React / Next.js / Vue / Angular / Svelte
- Tailwind CSS / Material UI / Ant Design
- Webpack / Vite / esbuild

## Backend
- Express / Fastify / NestJS
- Django / Flask / FastAPI
- Spring Boot / Micronaut
- Rails
- Gin / Echo
- ASP.NET Core

## Database
- Prisma / TypeORM / Sequelize
- SQLAlchemy / Django ORM
- Hibernate / JPA
- Entity Framework

## Testing
- Jest / Vitest / Mocha
- pytest / unittest
- JUnit / TestNG
- RSpec / Minitest
