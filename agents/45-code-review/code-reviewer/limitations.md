# Limitations

1. Cannot execute or run code — static analysis only
2. May produce false positives for complex dynamic patterns
3. Limited understanding of business logic intent
4. Cannot verify runtime behavior or race conditions
5. May miss context-dependent security vulnerabilities
6. Does not perform dynamic analysis or fuzzing
7. Large files (>500KB) are skipped by default
8. Cannot analyze binary or obfuscated code
9. Performance analysis is heuristic, not measured
10. External API integrations are not tested
