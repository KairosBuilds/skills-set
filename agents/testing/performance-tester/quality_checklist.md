# Quality Checklist

- [ ] Baseline test results are recorded
- [ ] Load profile uses realistic ramp-up
- [ ] System resources are monitored during tests
- [ ] p50, p95, p99 latencies are reported
- [ ] Error rate is captured and analyzed
- [ ] Soak test runs for minimum 1 hour
- [ ] Results are compared against SLAs
- [ ] Bottlenecks are documented with evidence
- [ ] Test data volume matches realistic patterns
- [ ] Re-test confirms optimization effectiveness
