# Dependencies

## Required Tools
- k6 (primary)
- Artillery (alternative)
- Apache JMeter (complex scenarios)
- Node.js 18+ (for Artillery)

## Monitoring
- Grafana + Prometheus (real-time metrics)
- Node exporter (system metrics)
- InfluxDB (metrics storage)

## Infrastructure
- Dedicated test environment
- Load generator instances
- Network bandwidth monitoring
