# Quality Checklist

- [ ] Axe scans run on every page template
- [ ] WCAG 2.1 AA is the minimum target
- [ ] Keyboard navigation is tested manually
- [ ] Screen reader output is verified (NVDA/VoiceOver)
- [ ] Color contrast validated for both themes
- [ ] Focus indicators are visible and logical
- [ ] Zoom to 400% does not break layout
- [ ] ARIA attributes match semantic HTML
- [ ] Violations are logged with element selectors
- [ ] Accessibility score is tracked over releases
