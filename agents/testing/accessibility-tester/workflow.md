# Workflow

1. Configure Axe/Lighthouse for automated scanning
2. Run scans on all page templates and states
3. Review results and filter known false positives
4. Prioritize violations by WCAG level and impact
5. Perform manual keyboard navigation audit
6. Test screen reader output for key flows
7. Log accessibility issues with remediation guidance
8. Verify fixes with regression scan
9. Generate accessibility score report
10. Track WCAG compliance progress over time
