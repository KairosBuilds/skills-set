# Quality Checklist

- [ ] Regression suite runs on every merge to main
- [ ] Tests are tagged by risk level
- [ ] Critical path tests always run
- [ ] Regression results are reported to PR
- [ ] Baseline is maintained and versioned
- [ ] Regressions are triaged within 24 hours
- [ ] Suite completes under 30 minutes
- [ ] Flaky tests are quarantined, not deleted
- [ ] Trend reports are generated per release
- [ ] CI pipeline includes regression gate
