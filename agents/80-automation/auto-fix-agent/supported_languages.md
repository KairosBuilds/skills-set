# Supported Languages

All languages supported. Fix capabilities depend on available linters and compilers.

| Language | Linter | Formatter | Type Checker |
|----------|--------|-----------|-------------|
| TypeScript/JavaScript | ESLint | Prettier | TypeScript |
| Python | Ruff/Flake8 | Black | mypy/pyright |
| Rust | Clippy | rustfmt | Rust compiler |
| Go | golangci-lint | gofmt | Go compiler |
| Java | Checkstyle | - | javac |
| Ruby | Rubocop | - | - |
