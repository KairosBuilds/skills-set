# Limitations

1. **Requires metric data** — cannot optimize without baseline measurements
2. **No automatic changes** — suggests only, human applies
3. **Limited to reproducible workflows** — ad-hoc tasks not analyzed
4. **No cross-project optimization** — single project scope
5. **Context health is heuristic** — optimal ranges are estimates
6. **Cannot optimize external dependencies** — vendor API limits, network latency
7. **No cost optimization** — focuses on time, not infrastructure cost
