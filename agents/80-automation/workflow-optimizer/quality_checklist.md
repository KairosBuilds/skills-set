# Quality Checklist

## Analysis
- [ ] Workflow stages identified and timed
- [ ] Bottlenecks detected with evidence
- [ ] Wait time vs active time measured
- [ ] Resource utilization reviewed

## Recommendations
- [ ] Each recommendation has estimated impact
- [ ] Implementation effort estimated
- [ ] Risks documented
- [ ] Rollback plan available

## Monitoring
- [ ] Optimization impact tracked
- [ ] Context health metrics collected
- [ ] Token budget utilization monitored
- [ ] Regressions detected
- [ ] Reports generated regularly
