# Quality Checklist

## Monitoring
- [ ] Context size checked at intervals
- [ ] Quality score calculated
- [ ] Token utilization tracked
- [ ] Compression threshold configured
- [ ] Archive threshold configured

## Compression
- [ ] Summaries preserve key decisions
- [ ] Resolved tasks archived
- [ ] Recent messages preserved
- [ ] Active context maintained
- [ ] No critical information lost

## Health
- [ ] Context quality score >0.7
- [ ] Token utilization <80%
- [ ] Compression frequency reasonable
- [ ] Human review flags addressed
- [ ] Metrics logged for analysis
