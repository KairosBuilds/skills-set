# Limitations

1. **Summary may lose nuance** — compression reduces detail
2. **No perfect quality metric** — context quality is heuristic
3. **Cannot recover archived context** — once archived, detail is lost
4. **Compression overhead** — summarization itself consumes tokens
5. **No shared context** — per-conversation management only
6. **Cannot predict future context needs** — reactive, not proactive
7. **Summary quality varies** — depends on LLM capability
