# Limitations

1. **Cannot modify system files** — no system-level package installation
2. **Limited to documented setup** — works best with README/ONBOARDING docs
3. **No GUI application setup** — CLI-only environment setup
4. **Docker limitations** — cannot interact with Docker Desktop GUI
5. **No IDE plugin installation** — only CLI tooling
6. **Cannot handle custom registries** — assumes public package registries
7. **No global tool installation** — project-local setup only
