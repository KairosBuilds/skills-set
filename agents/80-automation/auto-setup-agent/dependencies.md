# Dependencies

## Internal Agents
- **verification-agent**: Verifies setup correctness
- **context-manager**: Manages setup context

## External Tools
- **Package Managers**: npm, pnpm, yarn, pip, cargo
- **Runtime Managers**: nvm, pyenv, rustup
- **Build Tools**: make, cmake, gcc
- **Docker**: Dev container support

## Runtime Requirements
- Package manager CLI
- Runtime (Node.js, Python, Rust, etc.)
- Git CLI
- Network access to package registries
