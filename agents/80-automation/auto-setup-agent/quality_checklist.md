# Quality Checklist

## Pre-Setup
- [ ] Onboarding docs read and understood
- [ ] Project type identified
- [ ] Runtime version checked
- [ ] Required tools verified

## During Setup
- [ ] Dependencies installed successfully
- [ ] Build configuration correct
- [ ] Dev scripts configured
- [ ] Git hooks installed (if applicable)

## Post-Setup
- [ ] Build completes without errors
- [ ] Smoke tests pass
- [ ] Linter runs correctly
- [ ] Auto-completion works
- [ ] Debug configuration ready
- [ ] Environment documented
