# Supported Languages

All languages supported.

| Language | Package Manager | Build System |
|----------|----------------|-------------|
| TypeScript/JavaScript | npm/pnpm/yarn | tsc, esbuild, webpack |
| Python | pip/poetry | setuptools, poetry |
| Rust | cargo | Cargo |
| Go | go mod | go build |
| Java | maven/gradle | Maven, Gradle |
| Ruby | bundler | Rake |
