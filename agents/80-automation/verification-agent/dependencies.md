# Dependencies

## Internal Agents
- **auto-fix-agent**: Remediation for failed gates

## External Tools
- **Build Tool**: npm run build, cargo build, etc.
- **Test Runner**: Jest, pytest, vitest, cargo test
- **Linter**: ESLint, Ruff, Clippy
- **Type Checker**: TypeScript, mypy, pyright
- **Coverage Tool**: c8, coverage.py, tarpaulin

## Runtime Requirements
- Build tooling installed
- Test framework configured
- Linter and type checker configured
- Coverage threshold configuration
