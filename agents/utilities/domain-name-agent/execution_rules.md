# Execution Rules

## Name Generation
1. Start with user-provided keywords
2. Combine with prefixes/suffixes (get, go, hub, app, io, ly)
3. Use synonyms and related terms
4. Consider brandable words (portmanteaus, coined terms)
5. Generate short, memorable options

## Availability Checking
1. Query WHOIS for domain availability
2. Check preferred TLDs first (.com, .io, .dev)
3. Suggest alternative TLDs if primary is taken
4. Verify domain is not premium-priced
5. Check for trademark conflicts

## Quality Assessment
1. **Memorability**: Easy to spell and remember
2. **Length**: Shorter is better (<12 chars ideal)
3. **Pronounceability**: Easy to say over phone
4. **Brandability**: Distinctive and meaningful
5. **SEO potential**: Contains relevant keywords

## TLD Recommendations
1. .com for commercial/business
2. .io for tech/startups
3. .dev for developer tools
4. .ai for AI products
5. .app for applications
