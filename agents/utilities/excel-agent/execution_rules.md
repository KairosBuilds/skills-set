# Execution Rules

## Workbook Creation
1. Structure data with clear headers and consistent types
2. Apply formatting: bold headers, number formats, borders
3. Add data validation where appropriate
4. Use named ranges for dynamic references
5. Freeze header row for navigation

## Formula Usage
1. Use formulas for calculated fields (SUM, AVERAGE, IF, VLOOKUP)
2. Use structured references with tables
3. Avoid hardcoding values in formulas
4. Document complex formulas with comments
5. Test formulas with sample data

## Data Analysis
1. Create pivot tables for aggregation
2. Add pivot charts for visualization
3. Use conditional formatting for patterns
4. Sort and filter data for readability
5. Generate summary statistics

## Data Transformation
1. Clean data: remove duplicates, handle nulls
2. Convert data types appropriately
3. Merge multiple sheets or files
4. Split data into multiple columns
5. Standardize date/number formats
