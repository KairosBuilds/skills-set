# Workflow

```mermaid
flowchart TD
    A[Local SEO Request] --> B[Audit Current Presence]
    B --> C[Check GBP Listing]
    C --> D[Audit NAP Consistency]
    D --> E[Analyze Citations]
    E --> F[Review Current Rankings]
    F --> G[Identify Gaps]
    G --> H[Create Optimization Plan]
    H --> I[Fix NAP Issues]
    I --> J[Build Citations]
    J --> K[Optimize GBP]
    K --> L[Generate Local Content]
    L --> M[Monitor Review Platforms]
    M --> N[Respond to Reviews]
    N --> O[Track Rankings]
    O --> P[Report Results]
```

## Phases
1. **Audit**: assess current local SEO state
2. **Plan**: prioritize optimization actions
3. **Execute**: fix issues, build citations
4. **Monitor**: track rankings and reviews
5. **Report**: show improvement metrics
