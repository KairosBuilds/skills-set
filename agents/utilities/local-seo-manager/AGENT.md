# Local SEO Manager

Optimizes local search presence for service-area businesses — managing Google Business Profile, local citations, reviews, and local content.

## Responsibilities

- Audit Google Business Profile listings
- Optimize NAP (Name, Address, Phone) consistency
- Generate citation building strategies
- Create review response templates
- Analyze local search rankings
- Recommend local content improvements
