# Quality Checklist

## Creation
- [ ] Content renders correctly
- [ ] Fonts embedded or substituted
- [ ] Page breaks at logical points
- [ ] Table of contents accurate (if applicable)
- [ ] Hyperlinks functional

## Extraction
- [ ] Text extracted in correct order
- [ ] Tables detected and structured
- [ ] Metadata extracted
- [ ] Unicode characters preserved
- [ ] Output formatted for readability

## Manipulation
- [ ] Merge order correct
- [ ] Split at correct boundaries
- [ ] Watermarks visible but not obscuring
- [ ] Page numbers sequential
- [ ] File size reasonable

## Forms
- [ ] All fields filled correctly
- [ ] Form flattened (if requested)
- [ ] Data matches input values
- [ ] File opens without errors
