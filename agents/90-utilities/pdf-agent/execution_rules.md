# Execution Rules

## PDF Creation
1. Support creation from markdown, HTML, plain text
2. Auto-generate table of contents for multi-page docs
3. Apply consistent styling (fonts, margins, headers)
4. Support page numbering and section breaks
5. Embed fonts where needed

## Text Extraction
1. Extract text preserving reading order
2. Detect and extract tables (structured data)
3. Extract metadata (author, title, creation date)
4. Handle multi-column layouts
5. Preserve Unicode characters

## Document Manipulation
1. Merge multiple PDFs into single document
2. Split by page ranges, bookmarks, or file size
3. Rotate, reorder, delete pages
4. Add watermarks (text or image)
5. Add headers/footers to all pages

## Form Handling
1. Detect form fields automatically
2. Fill AcroForm fields programmatically
3. Flatten forms after filling (prevent editing)
4. Export form data as JSON/XML (XFDF)
