# Limitations

1. **No OCR support** — scanned PDFs require separate OCR tool
2. **Complex layout may lose formatting** — extraction may not preserve exact layout
3. **Large file performance** — very large PDFs may be slow
4. **Limited form field types** — some advanced form features unsupported
5. **No digital signatures** — cannot create or verify digital signatures
6. **Encrypted PDFs limited** — may not handle all encryption types
7. **No native image extraction** — embedded images handled via separate process
