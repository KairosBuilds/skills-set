# Quality Checklist

## Extraction
- [ ] Vendor name extracted correctly
- [ ] Date parsed in consistent format
- [ ] Amount extracted with correct currency
- [ ] Tax amount identified (if present)
- [ ] Missing fields flagged

## Organization
- [ ] Files renamed consistently
- [ ] Folder structure created
- [ ] Duplicate names handled
- [ ] Unrecognized files separated
- [ ] Original files preserved (copy not move)

## Reporting
- [ ] Summary CSV generated
- [ ] Monthly totals calculated
- [ ] Category totals accurate
- [ ] Annual summary generated
- [ ] Output validated against input
