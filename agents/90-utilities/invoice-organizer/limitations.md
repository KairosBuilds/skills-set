# Limitations

1. **OCR accuracy varies** — scanned documents may have errors
2. **No handwritten invoice support** — typed/printed only
3. **Format-dependent** — may not handle all invoice layouts
4. **No multi-currency conversion** — keeps original currency
5. **No accounting software integration** — CSV export only
6. **Cannot process password-protected PDFs** — encrypted files skipped
7. **No tax calculation** — uses extracted values as-is
