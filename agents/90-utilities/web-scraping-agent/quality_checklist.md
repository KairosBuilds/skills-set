# Quality Checklist

## Pre-Scrape
- [ ] robots.txt checked
- [ ] Rate limit configured
- [ ] Timeout set appropriately
- [ ] User-Agent configured

## Extraction
- [ ] Page fetched successfully
- [ ] Content extracted from main area
- [ ] HTML cleaned (scripts, styles removed)
- [ ] Structure preserved (headings, lists)
- [ ] Encoding handled correctly

## Post-Processing
- [ ] Output format correct
- [ ] Data validated for completeness
- [ ] No truncated content
- [ ] Error handling verified
- [ ] Output file/string returned
