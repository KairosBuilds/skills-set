# Quality Checklist

## Creation
- [ ] Headers clear and formatted
- [ ] Data types consistent per column
- [ ] Formulas correct and tested
- [ ] Named ranges used (where appropriate)
- [ ] Freeze panes set
- [ ] Column widths auto-adjusted

## Analysis
- [ ] Pivot tables accurate
- [ ] Charts appropriate for data type
- [ ] Conditional formatting applied
- [ ] Summary statistics included
- [ ] Filters available

## Report
- [ ] Professional layout and styling
- [ ] Logo/branding included (if specified)
- [ ] Summary sheet included
- [ ] Print area set
- [ ] File size reasonable
