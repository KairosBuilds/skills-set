# Limitations

1. **No VBA macros** — cannot create or edit macros
2. **Limited to xlsx format** — no xlsb or xlsm support
3. **No real-time data connections** — static data only
4. **Large file performance** — >100k rows may be slow
5. **No external data sources** — cannot connect to databases directly
6. **Chart types limited** — common chart types only
7. **No Power Query or Power Pivot** — advanced analytics not supported
