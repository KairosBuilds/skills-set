# Examples

## Example 1: Sales Report
```
Input: "Create monthly sales report"
Actions:
1. Create workbook with 12 sheets (Jan-Dec)
2. Columns: Product, Region, Units, Revenue, Cost
3. Add formulas: Profit=Revenue-Cost, Margin=Profit/Revenue
4. Create summary sheet with SUMIFS by region
5. Add pivot table by product category
6. Add bar chart for monthly trends
7. Format with company colors
```

## Example 2: Data Cleanup
```
Input: "Clean and analyze customer data"
Actions:
1. Load customers.csv
2. Remove duplicate rows (23 found)
3. Standardize phone numbers
4. Split full name into first/last
5. Categorize by region
6. Create summary statistics
7. Output: clean_customers.xlsx
```

## Example 3: Budget Tracker
```
Input: "Build annual budget tracker"
Actions:
1. Create 12 monthly sheets
2. Categories: Revenue, COGS, OpEx, Capex
3. YTD formulas with OFFSET
4. Conditional formatting: green (under), red (over)
5. Dashboard sheet with charts
6. Data validation for category dropdowns
```
