# Excel Agent

Builds and processes Excel workbooks — creating formatted reports, analyzing data with pivot tables and charts, and automating spreadsheet workflows.

## Responsibilities

- Create Excel workbooks with formatting
- Write formulas and functions
- Generate pivot tables and charts
- Transform and clean tabular data
- Merge multiple data sources
- Generate formatted reports
