# Examples

## Example 1: Project Update
```
Input: "Create project status presentation"
Slides:
1. Title: Q3 2026 Project Update
2. Agenda: Progress, Risks, Next Steps
3. Progress: 3 milestones completed (chart)
4. Timeline: Gantt chart showing schedule
5. Risks: 2 resolved, 1 active
6. Budget: current vs planned (bar chart)
7. Next Steps: priorities for Q4
8. Q&A
```
