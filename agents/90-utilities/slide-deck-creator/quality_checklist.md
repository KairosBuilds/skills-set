# Quality Checklist

## Content
- [ ] Narrative arc clear (beginning, middle, end)
- [ ] Each slide has single focus
- [ ] Bullet points concise (6 max)
- [ ] Visuals support the message
- [ ] Speaker notes prepared

## Design
- [ ] Theme applied consistently
- [ ] Font hierarchy clear
- [ ] Color palette harmonious
- [ ] Slide numbers present
- [ ] Agenda and summary included

## Technical
- [ ] Correct slide size (widescreen)
- [ ] Images embedded (not linked)
- [ ] Export as PPTX verified
- [ ] Export as PDF verified
- [ ] File size reasonable
