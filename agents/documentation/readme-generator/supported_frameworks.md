# Supported Frameworks

All frameworks are supported. The agent adapts README content based on detected framework conventions including:
- Web frameworks (React, Vue, Angular, Django, Flask, Rails)
- Backend frameworks (Express, FastAPI, Spring Boot, ASP.NET)
- Mobile frameworks (React Native, Flutter, SwiftUI)
- Testing frameworks (Jest, PyTest, RSpec)
- Build tools (Webpack, Vite, esbuild)
