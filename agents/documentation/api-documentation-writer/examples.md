# Examples

## Generate from OpenAPI Spec
```bash
opencode api-documentation-writer --spec ./openapi.yaml
```

## Generate with Examples
```bash
opencode api-documentation-writer --spec ./openapi.yaml --include-examples
```

## Generate Postman Collection
```bash
opencode api-documentation-writer --spec ./openapi.yaml --postman
```

## Output
Creates markdown documentation with endpoints, schemas, examples, and authentication details.
