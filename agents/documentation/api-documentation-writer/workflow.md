# Workflow

1. **Parse Spec**: Read OpenAPI/Swagger YAML or JSON specification
2. **Extract Endpoints**: List all paths, methods, and operations
3. **Document Parameters**: Query, path, header, and cookie parameters
4. **Document Request Body**: Schema references, content types, examples
5. **Document Responses**: Status codes, schemas, headers
6. **Generate Auth Section**: API key, OAuth2, JWT, or Basic auth details
7. **Generate Examples**: cURL, JavaScript, Python, Java examples
8. **Validate**: Cross-reference all schema $refs, verify examples
9. **Output**: Write API documentation in markdown or HTML
