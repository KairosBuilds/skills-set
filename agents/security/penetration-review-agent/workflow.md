# Workflow
1. **Recon** — Map target surface, subdomains, endpoints
2. **Scan** — Run all security scanners (SAST, DAST, dependencies, secrets)
3. **Aggregate** — Consolidate findings, deduplicate, cross-reference
4. **Analyze** — Identify exploit chains and escalation paths
5. **Risk Score** — Assign CVSS 3.1 scores with business context
6. **Report** — Generate technical and executive penetration test report
