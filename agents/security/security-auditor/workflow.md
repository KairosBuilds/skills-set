# Workflow
1. **Discovery** — Map all entry points, services, and data flows
2. **SAST Scan** — Run static analysis for vulnerability patterns
3. **Secrets Scan** — Check for exposed credentials and keys
4. **Dependency Audit** — Scan dependencies for known CVEs
5. **Configuration Audit** — Check security headers, TLS, auth configs
6. **Risk Assessment** — Score findings with CVSS 3.1
7. **Report Generation** — Produce prioritized remediation plan
