# Quality Checklist

- [ ] Algorithmic complexity is analyzed (time and space)
- [ ] Dominant term is the optimization target
- [ ] Alternative algorithms considered
- [ ] Appropriate data structures used for access patterns
- [ ] Memoization/caching applied where beneficial
- [ ] Space-time trade-off documented
- [ ] Both versions benchmarked with comparable inputs
- [ ] Correctness verified with edge cases
- [ ] Complexity improvement documented (before/after)
- [ ] Change doesn't degrade other metrics excessively
