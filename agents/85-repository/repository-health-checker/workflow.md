# Workflow

1. **Code Quality**: Lint results, complexity, test coverage
2. **Documentation**: README completeness, API docs, contributing guide
3. **CI/CD**: Pipeline status, recent failures, build times
4. **Security**: Vulnerabilities, secrets exposure, dependency audit
5. **Maintenance**: Commit frequency, response time to issues/PRs
6. **Dependencies**: Freshness, vulnerability status, license compliance
7. **Score**: Calculate per-dimension and overall health score
8. **Report**: Generate health report with prioritized recommendations
