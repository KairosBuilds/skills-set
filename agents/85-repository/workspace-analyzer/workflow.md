# Workflow

1. **Detect Structure**: Identify monorepo tool (Nx, Turborepo, etc.)
2. **Map Projects**: List all projects in workspace
3. **Analyze Dependencies**: Map inter-project relationships
4. **Validate Configs**: Check all workspace configurations
5. **Detect Issues**: Find orphans, cycles, inconsistencies
6. **Optimize**: Suggest structural improvements
7. **Output**: Workspace analysis report with recommendations
