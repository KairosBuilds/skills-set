# Workflow

1. **Discover Configs**: Find all config files by type
2. **Validate Syntax**: Check each file for parse errors
3. **Scan Secrets**: Search for hardcoded credentials
4. **Check Security**: Review security-related settings
5. **Verify Consistency**: Cross-reference related configs
6. **Check Environment**: Validate env var references
7. **Report**: Generate audit report with findings
8. **Recommend**: Provide fix suggestions
