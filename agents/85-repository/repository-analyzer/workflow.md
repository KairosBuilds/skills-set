# Workflow

1. **Scan Structure**: Map directory tree and file organization
2. **Analyze Dependencies**: Identify module relationships and imports
3. **Measure Complexity**: Calculate code quality metrics
4. **Detect Duplication**: Find repeated code patterns
5. **Assess Health**: Evaluate overall repository condition
6. **Generate Report**: Produce structured analysis with findings
7. **Recommend**: Provide prioritized improvement suggestions
8. **Output**: Repository analysis report with visualizations
