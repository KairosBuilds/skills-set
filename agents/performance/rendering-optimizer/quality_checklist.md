# Quality Checklist

- [ ] Performance profile recorded in target browser
- [ ] React Profiler identifies wasted renders
- [ ] Frame budget met (16ms for 60fps)
- [ ] No layout thrashing in requestAnimationFrame
- [ ] Long lists are virtualized
- [ ] Images are lazy loaded
- [ ] React.memo/useMemo applied where beneficial
- [ ] WebGL draw calls minimized
- [ ] Tested on target slowest device
- [ ] Before/after frame time comparison documented
