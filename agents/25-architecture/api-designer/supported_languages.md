# API Designer Supported Languages

API design is language-agnostic. Specifications cross all languages.

| Language | Support Level | Notes |
|----------|--------------|-------|
| TypeScript | Full | API client/server generation |
| JavaScript | Full | API implementation |
| Python | Full | API implementation, FastAPI |
| Go | Full | gRPC, REST implementation |
| Java | Full | Spring REST, gRPC |
| C# | Full | ASP.NET Web API, gRPC |
| Rust | Full | Actix, Axum REST, gRPC |
| Ruby | Full | Rails API, Grape |
| PHP | Full | Laravel API |
| Kotlin | Full | Ktor, Spring REST |
| YAML/JSON | Full | OpenAPI, Protobuf specs |
