# Event-Driven Architect Supported Languages

Event-driven architecture is language-agnostic at the design level.

| Language | Support Level | Notes |
|----------|--------------|-------|
| TypeScript | Full | KafkaJS, amqplib, MQTT |
| JavaScript | Full | KafkaJS, amqplib |
| Python | Full | confluent-kafka, pika, aiokafka |
| Java | Full | Kafka client, Spring Cloud Stream |
| Go | Full | Sarama, NATS client |
| C# | Full | Confluent .NET, NServiceBus |
| Rust | Full | rdkafka, lapin |
| Kotlin | Full | Kafka client, Spring Cloud Stream |
| SQL | Full | Stream processing queries |
