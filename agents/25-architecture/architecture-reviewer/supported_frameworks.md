# Architecture Reviewer Supported Frameworks

Architecture review is framework-agnostic. All frameworks are supported for architecture-level evaluation.

| Category | Frameworks | Support Level |
|----------|------------|--------------|
| Backend | Express, FastAPI, Spring Boot, ASP.NET Core, NestJS, Gin, Echo, Fiber, Django | Full |
| Frontend | React, Next.js, Vue, Nuxt, Angular, Svelte | Full |
| Mobile | React Native, Flutter, SwiftUI, Jetpack Compose | Full |
| Database | PostgreSQL, MySQL, MongoDB, DynamoDB, Cassandra, Neo4j, Redis | Full |
| Message Queue | Kafka, RabbitMQ, AWS SQS/SNS, Google Pub/Sub | Full |
| Cloud | AWS, Azure, GCP | Full |
| Containers | Docker, Kubernetes, ECS, Nomad | Full |
| API | REST, GraphQL, gRPC, WebSocket | Full |
