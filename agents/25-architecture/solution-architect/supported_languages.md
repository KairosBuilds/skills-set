# Solution Architect Supported Languages

The Solution Architect is language-agnostic for design and documentation tasks. All major programming languages are supported in architecture contexts.

| Language | Support Level | Notes |
|----------|--------------|-------|
| TypeScript | Full | Native design patterns, Node.js/Deno ecosystems |
| JavaScript | Full | Full ecosystem coverage |
| Python | Full | Web, data, ML architecture patterns |
| Go | Full | Microservices, concurrent systems |
| Java | Full | Enterprise, Spring ecosystem |
| C# | Full | .NET ecosystem, enterprise patterns |
| Rust | Full | Systems programming, performance-critical components |
| Ruby | Full | Rails ecosystem |
| PHP | Full | Laravel, WordPress ecosystems |
| Swift | Full | iOS/macOS architecture |
| Kotlin | Full | Android, JVM ecosystem |
| SQL | Full | Data architecture, query patterns |
| Shell/Bash | Full | Infrastructure, DevOps patterns |
| YAML/JSON | Full | Configuration, API specifications |
