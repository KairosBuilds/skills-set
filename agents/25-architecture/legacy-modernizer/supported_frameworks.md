# Legacy Modernizer Supported Frameworks

| Category | Legacy | Modern Target | Support |
|----------|--------|---------------|---------|
| Backend | J2EE, .NET Framework, Rails 3/4, Django 1.x, PHP 5.x | Spring Boot, ASP.NET Core, Rails 7, Django 4/5, Laravel 10/11, Express, FastAPI | Full |
| Frontend | jQuery, Backbone, AngularJS, Ember | React, Next.js, Vue 3, Angular 17+, Svelte | Full |
| Mobile | Objective-C, Swift < 5, Java Android | SwiftUI, Jetpack Compose, React Native, Flutter | Full |
| Database | MySQL 5.x, Oracle, SQL Server, MongoDB 3.x | PostgreSQL 15+, MySQL 8+, CockroachDB, AWS Aurora | Full |
| API | SOAP, XML-RPC, REST with XML | REST (JSON), GraphQL, gRPC | Full |
| Infrastructure | On-premise data centers, manual deployment | Cloud (AWS/Azure/GCP), Kubernetes, IaC (Terraform) | Full |
| Build | Ant, Make, Grunt, Gulp, Webpack 3 | Maven/Gradle, Vite, esbuild, Turbopack | Full |
| CI/CD | Jenkins, manual deployment | GitHub Actions, GitLab CI, ArgoCD | Full |
