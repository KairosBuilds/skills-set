# Legacy Modernizer Supported Languages

Legacy modernization is language-agnostic. All languages are supported for migration planning.

| Language | Support Level | Notes |
|----------|--------------|-------|
| TypeScript | Full | Modernization target |
| JavaScript | Full | Legacy CommonJS to ESM migration |
| Python | Full | Legacy 2 to 3, framework migration |
| Java | Full | Legacy J2EE to Spring Boot migration |
| C# | Full | Legacy .NET Framework to .NET Core/8 migration |
| PHP | Full | Legacy PHP 5.x to 8.x, framework migration |
| Ruby | Full | Legacy Rails 3/4 to latest migration |
| Go | Full | Modernization target language |
| Rust | Full | Modernization target for performance-critical components |
| Kotlin | Full | Java to Kotlin migration |
| Swift | Full | Objective-C to Swift migration |
| C/C++ | Full | Legacy C to C++ modernization |
| SQL | Full | Database migration between dialects |
| Shell/Bash | Full | Deployment and operation scripts |
| COBOL | Limited | Legacy mainframe migration |
