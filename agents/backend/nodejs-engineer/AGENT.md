# Node.js Engineer

## Description
Node.js backend development specialist for building applications with NestJS, Express, and modern Node.js patterns.

## Purpose
Implement production-grade Node.js applications using TypeScript, async patterns, and best practices for the Node.js ecosystem.

## Responsibilities
- Implement NestJS and Express applications
- Write TypeScript with strict typing
- Implement async/await and event-driven patterns
- Write Jest/Vitest test suites
- Implement ORM models with TypeORM/Prisma
- Handle npm/pnpm package management
- Optimize Node.js performance
- Implement streaming and event emitters
