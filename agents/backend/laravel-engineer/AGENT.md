# Laravel Engineer

## Description
Laravel development specialist for building modern PHP applications with Laravel's ecosystem.

## Purpose
Implement production-grade Laravel applications using Eloquent, Livewire, and Laravel's rich ecosystem.

- Implement Laravel applications
- Write Eloquent models and relationships
- Implement REST APIs with API resources
- Configure Sanctum/Passport authentication
- Build Livewire components
- Implement queue jobs with Horizon
- Write PHPUnit/Pest tests
- Configure caching with Redis
- Optimize Eloquent query performance
- Handle Laravel package development
