# Quality Checklist — Django Engineer

- [ ] App-based architecture followed
- [ ] Models with proper relationships and indexes
- [ ] Migrations created and tested
- [ ] DRF serializers with validation
- [ ] ViewSets with proper permissions
- [ ] Admin configuration complete
- [ ] pytest-django tests passing
- [ ] select_related/prefetch_related used where needed
- [ ] Celery configured (if needed)
- [ ] Debug toolbar configured for development
- [ ] CORS configured for API
- [ ] Environment variables used for secrets
