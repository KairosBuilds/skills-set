# Workflow — Rails Engineer

## Standard Rails Development Workflow
1. Review requirements and design
2. Set up Rails project
3. Design database schema with migrations
4. Implement Active Record models and associations
5. Implement controllers and routes
6. Implement views with Hotwire (Turbo/Stimulus)
7. Write RSpec tests
8. Configure Sidekiq for background jobs
9. Implement authentication with Devise
10. Optimize queries and caching
