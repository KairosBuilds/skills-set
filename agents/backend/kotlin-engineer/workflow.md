# Workflow — Kotlin Engineer

## Standard Kotlin Development Workflow
1. Review requirements and design
2. Set up Gradle Kotlin DSL project
3. Implement domain model (data classes, sealed classes)
4. Implement service layer with coroutines
5. Implement API endpoints (Ktor/Spring)
6. Write tests with kotlin.test
7. Add serialization configuration
8. Configure multiplatform modules (if needed)
