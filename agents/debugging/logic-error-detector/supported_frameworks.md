# Logic Error Detector — Supported Frameworks

All frameworks are supported. Logic error detection is algorithm-agnostic.

## Web Frameworks
- React, Angular, Vue, Svelte, Next.js, Nuxt, Express, Django, Flask, FastAPI, Spring Boot, ASP.NET, Rails, Laravel

## Mobile Frameworks
- React Native, Flutter, SwiftUI, UIKit, Android SDK

## Testing Frameworks
- Jest, Mocha, pytest, JUnit, RSpec

## State Management
- Redux, Vuex, Pinia, NgRx, Zustand, MobX
- Context APIs, Signals

## Database Frameworks
- Prisma, TypeORM, Sequelize, SQLAlchemy, Hibernate, Entity Framework
