# Runtime Error Specialist — Supported Frameworks

All frameworks are supported. Runtime errors are analyzed at the execution level.

## Web Frameworks
- Express, Django, Flask, FastAPI, Spring Boot, ASP.NET, Rails, Laravel, Next.js, Nuxt

## Mobile Frameworks
- React Native, Flutter, SwiftUI, UIKit, Android SDK

## ORM / Database
- Prisma, TypeORM, Sequelize, SQLAlchemy, Hibernate, Entity Framework
- Mongoose, ActiveRecord, Eloquent
