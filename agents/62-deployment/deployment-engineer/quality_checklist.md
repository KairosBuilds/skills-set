# Quality Checklist

- [ ] Deployment strategy matches application requirements
- [ ] Health checks are configured and validated
- [ ] Rollback procedure is documented and tested
- [ ] Container images are tagged with immutable versions
- [ ] Configuration is externalized from code
- [ ] Secrets are managed securely (not in plaintext)
- [ ] Resource limits are set for all containers
- [ ] Network policies are configured
- [ ] Monitoring and alerting are in place
- [ ] Deployment is logged and auditable
