# Workflow

1. **Analyze Application**: Determine runtime dependencies, ports, volumes
2. **Design Dockerfile**: Multi-stage builds, dependency installation
3. **Create Dockerfile**: Write optimized Dockerfile with best practices
4. **Configure Compose**: Create docker-compose.yml for multi-service setup
5. **Optimize Image**: Reduce image size, use layer caching
6. **Security Scan**: Scan for vulnerabilities with Trivy or Docker Scout
7. **Build & Test**: Build image locally and verify functionality
8. **Document**: Record image structure, ports, volumes, and environment variables
