# Execution Rules

1. Define resource requests and limits for all containers
2. Use namespaces to isolate environments
3. Enable RBAC with least-privilege principle
4. Configure pod security contexts (non-root, read-only root fs)
5. Use Helm for package management and releases
6. Implement readiness and liveness probes
7. Enable NetworkPolicies for micro-segmentation
8. Use ConfigMaps and Secrets for configuration
